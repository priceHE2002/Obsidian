## 【第一板块：时域离散信号与系统基础】

### 习题 1-1

【题目】

计算下列序列的卷积和 $y[k]=x[k]*g[k]$：

1. $x[k]=\alpha^k u[k], \quad g[k]=\beta^k u[k]$
    
2. $x[k]=g[k]=R_4[k] = \begin{cases} 1, & 0 \le k \le 3 \\ 0, & \text{其他} \end{cases}$
    

【详细解答】

(1) 根据卷积定义：$y[k] = \sum_{n=-\infty}^{\infty} x[n]g[k-n]$。

代入表达式：

$$y[k] = \sum_{n=-\infty}^{\infty} \alpha^n u[n] \beta^{k-n} u[k-n]$$

确定求和区间：

- $u[n] \neq 0 \Rightarrow n \ge 0$
    
- $u[k-n] \neq 0 \Rightarrow k-n \ge 0 \Rightarrow n \le k$
    

两者的公共区间为 $0 \le n \le k$。

- 若 $k<0$，区间为空，$y[k]=0$。
    
- 当 $k \ge 0$ 时：$$y[k] = \sum_{n=0}^{k} \alpha^n \beta^{k-n} = \beta^k \sum_{n=0}^{k} (\frac{\alpha}{\beta})^n$$
    

这是一个等比数列求和，公比 $q = \alpha/\beta$。必须分情况讨论：

- 情况一：$\alpha \neq \beta$  
    利用公式 $\sum_{n=0}^{N} q^n = \frac{1-q^{N+1}}{1-q}$：$$y[k] = \beta^k \frac{1-(\frac{\alpha}{\beta})^{k+1}}{1-\frac{\alpha}{\beta}} = \beta^k \frac{\frac{\beta^{k+1}-\alpha^{k+1}}{\beta^{k+1}}}{\frac{\beta-\alpha}{\beta}} = \frac{\beta^{k+1} - \alpha^{k+1}}{\beta - \alpha}$$  
    综合写为：$$y[k] = \frac{\beta^{k+1} - \alpha^{k+1}}{\beta - \alpha} u[k]$$
    
- 情况二：$\alpha = \beta$  
    求和项变为 $\sum_{n=0}^{k} 1 = k+1$。$$y[k] = (k+1)\alpha^k u[k]$$
    

(2) $x[k]$ 和 $g[k]$ 都是长度为 $N=4$ 的矩形序列 $\{1, 1, 1, 1\}$。

卷积后的长度 $L = N_1 + N_2 - 1 = 4 + 4 - 1 = 7$。

利用**列表法（对位相乘求和）**计算 $y[k] = \sum_{n=0}^{3} x[n]x[k-n]$：

- $k=0$: $1\times1 = 1$
    
- $k=1$: $1\times1 + 1\times1 = 2$
    
- $k=2$: $1\times1 + 1\times1 + 1\times1 = 3$
    
- $k=3$: $1\times1 + 1\times1 + 1\times1 + 1\times1 = 4$
    
- $k=4$: $1\times1 + 1\times1 + 1\times1 = 3$ (滑出1个)
    
- $k=5$: $1\times1 + 1\times1 = 2$
    
- $k=6$: $1\times1 = 1$
    

结果：$y[k] = \{1, 2, 3, 4, 3, 2, 1\}, \quad k=0,1,\dots,6$。

【知识点深度剖析与扩展】

- 二级结论（常用卷积公式）：
    

- 单边指数序列卷积： $\alpha^k u[k] * \beta^k u[k] = \frac{\beta^{k+1}-\alpha^{k+1}}{\beta-\alpha}u[k] \quad (\alpha \neq \beta)$。
    
- 重根情况： $\alpha^k u[k] * \alpha^k u[k] = (k+1)\alpha^k u[k]$。这对应 $z$ 域的二阶极点 $(1-\alpha z^{-1})^{-2}$。
    
- 矩形卷积： 两个长度分别为 $N$ 和 $M$ 的矩形序列 $R_N[k] * R_M[k]$，结果为梯形序列（当 $N \neq M$）或三角形序列（当 $N=M$）。长度为 $N+M-1$。
    

- 其他方案（Z变换法）： 对于(1)，$Y(z) = X(z)G(z) = \frac{1}{(1-\alpha z^{-1})(1-\beta z^{-1})}$。做部分分式展开后求逆变换，可得相同结果。这种方法在系统分析中更通用。
    
- 易错点： 忘记讨论 $\alpha=\beta$ 的情况。卷积后序列长度的计算：$L = N_1 + N_2 - 1$。
    

### 习题 1-2

【题目】

1. 已知序列 $x[k]=\{1,2,3,4\}$，$y[k]=\{-1,1,-2,3\}$，计算自相关函数 $r_x[n]$，互相关函数 $r_{xy}[n]$ 和 $r_{yx}[n]$。
    
2. 若两个序列的自相关函数相同，这两个序列有何关系？
    
3. 若信号 $s[k]=x[k]+n[k]$，且 $x[k]$ 与 $n[k]$ 不相关，如何判断 $s[k]$ 中存在 $x[k]$？
    

【详细解答】

(1) 公式：$r_{xy}[n] = \sum_{k=-\infty}^{\infty} x[k]y[k+n]$。

- 自相关 $r_x[n]$： $x[k]=\{1,2,3,4\}$。
    

- $n=0$: $\sum x[k]^2 = 1+4+9+16 = 30$。
    
- $n=1$: $1\times2 + 2\times3 + 3\times4 = 20$。
    
- $n=2$: $1\times3 + 2\times4 = 11$。
    
- $n=3$: $1\times4 = 4$。
    
- 利用偶对称性 $r_x[n]=r_x[-n]$：$r_x[n] = \{4, 11, 20, \underline{30}, 20, 11, 4\}$。
    

- 互相关 $r_{xy}[n]$： $y[k]=\{-1, 1, -2, 3\}$。
    

- $n=0$: $1(-1)+2(1)+3(-2)+4(3) = 7$。
    
- $n=1$: $1(1)+2(-2)+3(3) = 6$。
    
- $n=2$: $1(-2)+2(3)=4$。
    
- $n=3$: $1(3)=3$。
    
- $n=-1$: $x[1]y[0]+x[2]y[1]+x[3]y[2] = 2(-1)+3(1)+4(-2) = -7$。
    
- $n=-2$: $x[2]y[0]+x[3]y[1] = 3(-1)+4(1) = 1$。
    
- $n=-3$: $x[3]y[0] = 4(-1) = -4$。
    
- $r_{xy}[n] = \{-4, 1, -7, \underline{7}, 6, 4, 3\}$。
    

- 互相关 $r_{yx}[n]$： 利用 $r_{yx}[n] = r_{xy}[-n]$（翻转）。
    

- $r_{yx}[n] = \{3, 4, 6, \underline{7}, -7, 1, -4\}$。
    

(2) 若 $r_x[n] = r_y[n]$，说明两序列具有相同的幅度谱（功率谱），但相位谱可能不同。例如 $x[k]$ 和它的时反序列、移位序列具有相同的自相关。

(3) 计算 $s[k]$ 的自相关函数：

$$r_s[n] = r_x[n] + r_n[n] + r_{xn}[n] + r_{nx}[n]$$

由于 $x, n$ 不相关，$r_{xn} \approx 0, r_{nx} \approx 0$。如果 $n[k]$ 是白噪声，其自相关 $r_n[n] = \sigma^2 \delta[n]$（近似仅在 $n=0$ 处有值）。因此，在 $n \neq 0$ 处，$r_s[n] \approx r_x[n]$。若检测到显著的非零相关值（特别是周期性峰值），则说明存在信号 $x[k]$。

【知识点深度剖析与扩展】

- 二级结论：
    

- 相关与卷积的关系：$r_{xy}[n] = x[n] * y[-n]$。自相关 $r_x[n] = x[n] * x[-n]$。
    
- 自相关性质：$r_x[n]$ 是偶函数；$r_x[0]$ 等于信号能量；$|r_x[n]| \le r_x[0]$。
    

- 频域关系： $R_x(e^{j\Omega}) = |X(e^{j\Omega})|^2$（功率谱密度/能量谱密度）。这解释了为什么自相关相同意味着幅度谱相同。
    
- 其他方案（表格法）： 计算互相关时，可以将 $x[k]$ 不动，将 $y[k]$ 向左移（正 $n$）或向右移（负 $n$），对应位置相乘相加。这与卷积的“翻转平移”不同，相关不翻转。
    

### 习题 1-3

【题目】

已知序列 $x[k]=\alpha^k u[k]$，其中 $|\alpha|<1$，求其自相关函数 $r_x[n]$。

【详细解答】

$$r_x[n] = \sum_{k=-\infty}^{\infty} x[k]x[k+n] = \sum_{k=-\infty}^{\infty} \alpha^k u[k] \alpha^{k+n} u[k+n]$$

利用自相关的偶对称性 $r_x[n]=r_x[-n]$，我们只计算 $n \ge 0$ 的情况。

当 $n \ge 0$ 时：

- $u[k]$ 非零区间：$k \ge 0$。
    
- $u[k+n]$ 非零区间：$k \ge -n$。
    
- 重叠区间为 $k \ge 0$。
    

$$r_x[n] = \sum_{k=0}^{\infty} \alpha^k \alpha^{k+n} = \alpha^n \sum_{k=0}^{\infty} (\alpha^2)^k$$

利用无穷等比求和公式（$|\alpha|<1 \Rightarrow |\alpha^2|<1$）：

$$r_x[n] = \alpha^n \frac{1}{1-\alpha^2}, \quad n \ge 0$$

结合对称性，将 $n$ 替换为 $|n|$：

$$\mathbf{r_x[n] = \frac{1}{1-\alpha^2} \alpha^{|n|}, \quad -\infty < n < \infty}$$

【知识点深度剖析与扩展】

- 二级结论（常用变换对）： $\alpha^{|n|} \leftrightarrow \frac{1-\alpha^2}{1-2\alpha\cos\Omega+\alpha^2}$ （DTFT对）。本题结果表明，一阶AR模型（自回归过程）的自相关函数是双边指数衰减的。
    
- 其他方案（Z变换法）： 利用 $R_x(z) = X(z)X(z^{-1})$。
    

- $X(z) = \frac{1}{1-\alpha z^{-1}}$，$X(z^{-1}) = \frac{1}{1-\alpha z}$。
    
- $R_x(z) = \frac{1}{(1-\alpha z^{-1})(1-\alpha z)} = \frac{-1/\alpha \cdot z^{-1}}{(1-\alpha z^{-1})(1-\frac{1}{\alpha}z^{-1})}$。
    
- 通过部分分式展开求逆变换，同样可得 $\frac{1}{1-\alpha^2}\alpha^{|n|}$。
    

### 习题 1-4

【题目】

判断下列系统是否为(1)线性; (2)因果; (3)非时变; (4)稳定。

(a) $y[k]=k^2 x[k]$

(b) $y[k]=a + \sum_{n=-3}^{3} x[k-n], \quad a \ne 0$

【详细解答】

(a) $y[k]=k^2 x[k]$

- 线性：满足。$T[ax_1+bx_2] = k^2(ax_1+bx_2) = a(k^2 x_1) + b(k^2 x_2)$。
    
- 因果：满足。$y[k]$ 只取决于当前时刻输入 $x[k]$。
    
- 非时变（时变）：不满足。系数 $k^2$ 随时间变化。输入延迟 $x[k-N]$，输出为 $k^2 x[k-N]$；而原输出延迟为 $(k-N)^2 x[k-N]$，两者不等。
    
- 稳定：不满足。若输入为有界常数 $1$，输出 $k^2$ 随 $k \to \infty$ 发散。
    

(b) $y[k]=a + \sum_{n=-3}^{3} x[k-n]$

- 线性：不满足。常数 $a \neq 0$ 破坏了零输入零输出特性（输入为0，输出为 $a$）。
    
- 因果：不满足。展开式包含 $x[k+3]$，需要未来的输入。
    
- 非时变（时不变）：满足。系数为常数，且求和结构不随时间改变。
    
- 稳定：满足。若 $|x[k]| < M$，则 $|y[k]| \le |a| + 7M < \infty$。
    

【知识点深度剖析与扩展】

- 判定技巧（二级结论）：
    

- 线性：方程中不能出现 $x^2, |x|, \cos(x)$ 等非线性运算，也不能有非零常数项（如 $a$）。
    
- 时不变：系数不能显式包含 $k$（如 $k \cdot x[k]$），不能有时间缩放（如 $x[2k]$）。
    
- 因果：输出不依赖于 $x[k+n]$ ($n>0$)。
    
- 稳定：有界输入产生有界输出 (BIBO)。充要条件是 $\sum |h[k]| < \infty$。
    

- 增量线性系统：对于(b)，如果去掉 $a$，则变为线性。这种带偏置的系统称为“增量线性系统”。
    

### 习题 1-5

【题目】

已知某离散LTI系统的单位脉冲响应 $h[k]=(1/3)^k u[k]$，求下列输入下的零状态响应：

(1) $x[k]=u[k]$

(2) $x[k]=u[k]-u[k-N]$

(3) $x[k]=(1/2)^k u[k]$

【详细解答】

(1) $x[k]=u[k]$（求阶跃响应）

$$y[k] = x[k]*h[k] = \sum_{n=0}^{k} (\frac{1}{3})^n \cdot 1 = \frac{1-(1/3)^{k+1}}{1-1/3} u[k] = \frac{3}{2}[1-(\frac{1}{3})^{k+1}]u[k]$$

(2) $x[k]=u[k]-u[k-N]$

利用线性时不变性，设 (1) 中的阶跃响应为 $s[k]$。输入是 $u[k]$ 和延迟 $N$ 的 $u[k-N]$ 的差。

$$y[k] = s[k] - s[k-N] = \frac{3}{2}[1-(\frac{1}{3})^{k+1}]u[k] - \frac{3}{2}[1-(\frac{1}{3})^{k-N+1}]u[k-N]$$

(3) $x[k]=(1/2)^k u[k]$

这是两个不同底数指数序列的卷积，套用 1-1 题公式 ($\alpha=1/2, \beta=1/3$)：

$$y[k] = \frac{(1/3)^{k+1} - (1/2)^{k+1}}{1/3 - 1/2} u[k] = \frac{(1/3)(1/3)^k - (1/2)(1/2)^k}{-1/6} u[k]$$

化简得：

$$y[k] = [3(\frac{1}{2})^k - 2(\frac{1}{3})^k]u[k]$$

【知识点深度剖析与扩展】

- 二级结论：
    

- 一阶系统阶跃响应：$a^k u[k]$ 的阶跃响应为 $\frac{1-a^{k+1}}{1-a}u[k]$。
    
- 响应模态：输出响应包含两部分模态——输入模态（由 $x[k]$ 决定，如 $1/2$）和系统自然模态（由 $h[k]$ 极点决定，如 $1/3$）。系数大小取决于两者在 $z$ 平面上的距离。
    

- 其他方案： Z变换法：$Y(z) = H(z)X(z) = \frac{1}{1-\frac{1}{3}z^{-1}} \cdot \frac{1}{1-\frac{1}{2}z^{-1}}$。部分分式展开为 $\frac{A}{1-\frac{1}{3}z^{-1}} + \frac{B}{1-\frac{1}{2}z^{-1}}$，求逆变换即可。这种方法在阶数较高时更高效。
    

### 习题 1-6

【题目】

已知某离散LTI系统的单位阶跃响应为 $s[k]$，求该系统对任意输入 $x[k]$ 的响应。

【详细解答】

单位脉冲与阶跃的关系：$\delta[k] = u[k] - u[k-1]$。

根据 LTI 特性，单位脉冲响应 $h[k]$ 为：

$$h[k] = s[k] - s[k-1]$$

任意输入的响应为输入与 $h[k]$ 的卷积：

$$y[k] = x[k] * h[k] = x[k] * (s[k] - s[k-1])$$

或者展开写：$y[k] = \sum_{n=-\infty}^{\infty} x[n] (s[k-n] - s[k-n-1])$。

【知识点深度剖析与扩展】

- 二级结论（差分性质）： 线性时不变系统与差分运算（$\nabla$）是可交换的。即 $y[k] = x[k] * \nabla s[k] = \nabla (x[k] * s[k])$。
    
- 物理意义： 在实际工程中，产生理想的冲激信号很难，但产生阶跃信号（开关）很容易。通过测量阶跃响应再求差分，是获取系统特性的常用手段。
    

### 习题 1-7

【题目】

根据系统方框图（描述结构），求系统的单位脉冲响应 $h[k]$。

结构描述： $x[k]$ 分两路，一路经过 $h_1$；另一路经过 $h_2$ 与 ($h_3$ 串联 $h_4$) 并联，然后再串联 $h_5$。最后两路相加得到 $y[k]$。

【详细解答】

根据系统的连接关系：

1. $h_3$ 与 $h_4$ 串联 $\rightarrow$ 等效为 $h_3[k] * h_4[k]$。
    
2. 上述结果与 $h_2$ 并联 $\rightarrow$ 等效为 $h_2[k] + h_3[k] * h_4[k]$。
    
3. 该部分与 $h_5$ 串联 $\rightarrow$ 等效为 $(h_2[k] + h_3[k] * h_4[k]) * h_5[k]$。
    
4. 最后与 $h_1$ 并联 $\rightarrow$ 总响应：
    

$$h[k] = h_1[k] + h_2[k]*h_5[k] + h_3[k]*h_4[k]*h_5[k]$$

【知识点深度剖析与扩展】

- 核心法则：
    

- 串联 (Cascade) $\Leftrightarrow$ 时域卷积 $\Leftrightarrow$ Z域乘积 ($H_1(z)H_2(z)$)。
    
- 并联 (Parallel) $\Leftrightarrow$ 时域相加 $\Leftrightarrow$ Z域相加 ($H_1(z)+H_2(z)$)。
    
- 反馈 (Feedback) $\Leftrightarrow$ $\frac{H_1(z)}{1 \mp H_1(z)H_2(z)}$ (注意正负反馈符号)。
    

- 扩展： 如果将题中卷积换成Z变换 $H(z)$，式子形式完全一样，计算会由积分变为代数运算。
    

## 【第二板块：频域分析（DFS/DTFT）】

### 习题 1-8

【题目】

试求下列离散周期信号的频谱（DFS）：

(1) $\tilde{x}_1[k]=\{\dots, 1, 2, 0, 2, \dots\}$ (周期 $N=4$)

(2) $\tilde{x}_2[k]=\sin(\pi k/4)$

【详细解答】

(1) $N=4$。

公式：$\tilde{X}[m] = \sum_{k=0}^{3} \tilde{x}[k] W_4^{mk}, \quad W_4 = e^{-j\pi/2} = -j$。

- $m=0$: $\sum x = 1+2+0+2 = 5$
    
- $m=1$: $1(1) + 2(-j) + 0 + 2(j) = 1$
    
- $m=2$: $1(1) + 2(-1) + 0 + 2(-1) = -3$
    
- $m=3$: $1(1) + 2(j) + 0 + 2(-j) = 1$
    

结果：$\tilde{X}[m] = \{5, 1, -3, 1\}$。

(2) 周期 $N = \frac{2\pi}{\pi/4} = 8$。利用欧拉公式展开：

$$\tilde{x}_2[k] = \frac{1}{2j} (e^{j\frac{2\pi}{8}k} - e^{-j\frac{2\pi}{8}k})$$

对比 IDFS 公式 $\tilde{x}[k] = \frac{1}{N} \sum \tilde{X}[m] e^{j\frac{2\pi}{N}mk}$，这里 $N=8$。

- $e^{j\frac{2\pi}{8}k}$ 项系数为 $\frac{1}{2j}$，对应 $m=1$。所以 $\frac{1}{8}\tilde{X}[1] = \frac{1}{2j} \Rightarrow \tilde{X}[1] = -4j$。
    
- $e^{-j\frac{2\pi}{8}k} = e^{j\frac{2\pi}{8}7k}$ 项系数为 $-\frac{1}{2j}$，对应 $m=7$。所以 $\frac{1}{8}\tilde{X}[7] = -\frac{1}{2j} \Rightarrow \tilde{X}[7] = 4j$。
    

结果：在一个周期内，$\tilde{X}[1]=-4j, \tilde{X}[7]=4j$，其余为0。

【知识点深度剖析与扩展】

- 二级结论（常用DFS对）：
    

- $\cos(\frac{2\pi}{N}k) \leftrightarrow \frac{N}{2}(\delta[m-1] + \delta[m-(N-1)])$。
    
- $\sin(\frac{2\pi}{N}k) \leftrightarrow \frac{N}{2j}(\delta[m-1] - \delta[m-(N-1)])$。
    

- 其他方案（矩阵法）： $\mathbf{X} = \mathbf{W} \mathbf{x}$，其中 $\mathbf{W}$ 是 DFT 矩阵。适合计算机求解。
    

### 习题 1-9

【题目】

已知周期序列 $\tilde{x}[k]=\{\dots, 0, 1, 0, 2, \dots\}$ 和 $\tilde{h}[k]=\{\dots, 2, 0, 2, 1, \dots\}$，周期 $N=4$，求周期卷积 $\tilde{y}[k]$。

【详细解答】

利用矩阵法（循环卷积矩阵）：

$$\begin{bmatrix} \tilde{y}[0] \\ \tilde{y}[1] \\ \tilde{y}[2] \\ \tilde{y}[3] \end{bmatrix} = \begin{bmatrix} 2 & 1 & 2 & 0 \\ 0 & 2 & 1 & 2 \\ 2 & 0 & 2 & 1 \\ 1 & 2 & 0 & 2 \end{bmatrix} \begin{bmatrix} 0 \\ 1 \\ 0 \\ 2 \end{bmatrix} = \begin{bmatrix} 1\times1 + 0\times2 = 1 \\ 2\times1 + 2\times2 = 6 \\ 0\times1 + 1\times2 = 2 \\ 2\times1 + 2\times2 = 6 \end{bmatrix}$$

结果：$\tilde{y}[k] = \{\dots, 1, 6, 2, 6, \dots\}$。

【知识点深度剖析与扩展】

- 二级结论（循环卷积矩阵）： 矩阵的第一列是 $\tilde{h}[k]$，第二列是 $\tilde{h}[k]$ 循环右移一位，以此类推。这称为循环矩阵 (Circulant Matrix)。
    
- 其他方案：
    

- 同心圆法：一个序列逆时针排列，另一个顺时针排列，旋转相乘相加。
    
- DFT法：$\tilde{Y}[m] = \tilde{X}[m] \tilde{H}[m]$，先求两者的 DFS，对应相乘，再求 IDFS。这是快速卷积的基础。
    

### 习题 1-10

【题目】

求下列非周期序列的 DTFT，并画出幅度谱：

(1) $x_1[k] = \{0.5, 1, 0.5\}$ (中心在0)

(2) $x_2[k] = \{-1, -2, 6, -2, -1\}$ (中心在0)

(3) $x_3[k] = u[k] - u[k-N]$

【详细解答】

(1) $X_1(e^{j\Omega}) = 0.5 e^{j\Omega} + 1 + 0.5 e^{-j\Omega} = 1 + \cos\Omega$。

(2) $X_2(e^{j\Omega}) = -1(e^{j2\Omega}+e^{-j2\Omega}) - 2(e^{j\Omega}+e^{-j\Omega}) + 6 = 6 - 4\cos\Omega - 2\cos(2\Omega)$。

(3) 这是矩形窗序列。利用等比数列求和：

$$X_3(e^{j\Omega}) = \sum_{k=0}^{N-1} e^{-j\Omega k} = \frac{1-e^{-jN\Omega}}{1-e^{-j\Omega}} = \frac{e^{-jN\Omega/2}(e^{jN\Omega/2}-e^{-jN\Omega/2})}{e^{-j\Omega/2}(e^{j\Omega/2}-e^{-j\Omega/2})} = e^{-j\frac{N-1}{2}\Omega} \frac{\sin(N\Omega/2)}{\sin(\Omega/2)}$$

幅度谱为 $|\frac{\sin(N\Omega/2)}{\sin(\Omega/2)}|$。

【知识点深度剖析与扩展】

- 二级结论（常用DTFT对）：
    

- 矩形窗 (Rectangular Window)： $R_N[k] \leftrightarrow e^{-j\frac{N-1}{2}\Omega} \frac{\sin(N\Omega/2)}{\sin(\Omega/2)}$。
    
- Sinc函数： $\frac{\sin(\omega_c k)}{\pi k} \leftrightarrow$ 理想低通频谱（矩形谱）。
    

- 性质应用：
    

- 实偶对称序列 $\leftrightarrow$ 实偶对称频谱（如题1、2）。
    
- 线性相位： 如果序列关于中心对称，其频谱具有线性相位项 $e^{-j\alpha\Omega}$。
    

### 习题 1-11

【题目】

已知 $x[k]$ 的频谱为 $X(e^{j\Omega})$，画出 $y[k] = x[k]\cos(0.8\pi k)$ 的频谱。

【详细解答】

利用调制（频移）特性：

$$y[k] = x[k] \frac{1}{2}(e^{j0.8\pi k} + e^{-j0.8\pi k})$$$$Y(e^{j\Omega}) = \frac{1}{2} [ X(e^{j(\Omega-0.8\pi)}) + X(e^{j(\Omega+0.8\pi)}) ]$$

### 习题 1-12

【题目】

利用 DTFT 性质求频谱：

(1) $x_1[k]=\alpha^k u[-k], \quad |\alpha|>1$

(2) $x_2[k]=\alpha^k u[k-3], \quad |\alpha|<1$

(3) $x_3[k]=(k+1)\alpha^k u[k], \quad |\alpha|<1$

(4) $x_4[k]=\sum_{n=0}^{\infty}(1/4)^{k}\delta[k-3n]$

【详细解答】

(1) 令 $g[k] = \alpha^{-k} u[k] = (\alpha^{-1})^k u[k]$。

因 $|\alpha|>1$，故 $|\alpha^{-1}|<1$，DTFT 存在。

$G(e^{j\Omega}) = \frac{1}{1-\alpha^{-1}e^{-j\Omega}}$。

$x_1[k] = g[-k]$，利用时域翻转特性：

$X_1(e^{j\Omega}) = G(e^{-j\Omega}) = \frac{1}{1-\alpha^{-1}e^{j\Omega}}$。

(2) $x_2[k] = \alpha^3 \cdot \alpha^{k-3} u[k-3]$。

利用时移特性 $x[k-n_0] \leftrightarrow e^{-j\Omega n_0}X(e^{j\Omega})$：

$X_2(e^{j\Omega}) = \alpha^3 e^{-j3\Omega} \frac{1}{1-\alpha e^{-j\Omega}}$。

(3) $x_3[k] = k\alpha^k u[k] + \alpha^k u[k]$。

利用频域微分特性 $k g[k] \leftrightarrow j \frac{d G}{d\Omega}$。

$G(e^{j\Omega}) = \frac{1}{1-\alpha e^{-j\Omega}}$。

导数部分：$j \frac{d}{d\Omega} ((1-\alpha e^{-j\Omega})^{-1}) = j (-1)(1-\alpha e^{-j\Omega})^{-2} (- \alpha (-j) e^{-j\Omega}) = \frac{\alpha e^{-j\Omega}}{(1-\alpha e^{-j\Omega})^2}$。

相加得：

$X_3(e^{j\Omega}) = \frac{\alpha e^{-j\Omega} + (1-\alpha e^{-j\Omega})}{(1-\alpha e^{-j\Omega})^2} = \frac{1}{(1-\alpha e^{-j\Omega})^2}$。

(4) 序列仅在 $k=0, 3, 6, \dots$ 处有值 $(1/4)^{3n}$。

$X_4(e^{j\Omega}) = \sum_{n=0}^{\infty} (1/4)^{3n} e^{-j\Omega (3n)} = \sum_{n=0}^{\infty} (\frac{e^{-j3\Omega}}{64})^n = \frac{1}{1 - \frac{1}{64}e^{-j3\Omega}}$。

【知识点深度剖析与扩展】

- 常用变换对： $(n+1)a^n u[n] \leftrightarrow \frac{1}{(1-ae^{-j\Omega})^2}$。这一结论可以直接由第(3)题结果验证，值得记忆。
    
- 频域微分： 乘 $k$ 对应频域微分，乘 $k^2$ 对应二次微分。这在求斜坡响应或群时延时非常有用。
    
- 插值性质： 第(4)题展示了内插 (Upsampling) 的频域效果。若 $y[k] = x[k/M]$ (当 $k$ 是 $M$ 的倍数)，则 $Y(e^{j\Omega}) = X(e^{jM\Omega})$。这里是先构建 $(1/4)^n$ 序列，再 3倍内插。
    

### 习题 1-13

【题目】

已知有限长序列 $x[k]=\{-1, 1, 2, 0, 3, -2\}$ (k=-1到4)，不计算 DTFT，求：

(1) $X(e^{j0})$

(2) $X(e^{j\pi})$

(3) $\int_{-\pi}^{\pi} X(e^{j\Omega}) d\Omega$

(4) $\int_{-\pi}^{\pi} |X(e^{j\Omega})|^2 d\Omega$

【详细解答】

(1) $X(e^{j0}) = \sum x[k] e^{-j0} = \sum x[k] = -1+1+2+0+3-2 = 3$。

(2) $X(e^{j\pi}) = \sum x[k] e^{-j\pi k} = \sum x[k](-1)^k = (-1)(-1)^{-1} + 1(1) + 2(-1) + 0 + 3(-1)^4 + (-2)(-1)^5 = 1 + 1 - 2 + 0 + 3 + 2 = 5$。

(3) 根据 IDTFT 定义：$x[k] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\Omega}) e^{j\Omega k} d\Omega$。

令 $k=0$，则 $\int_{-\pi}^{\pi} X(e^{j\Omega}) d\Omega = 2\pi x[0] = 2\pi(1) = 2\pi$。

(4) 根据帕塞瓦尔定理：$\int_{-\pi}^{\pi} |X|^2 d\Omega = 2\pi \sum |x[k]|^2 = 2\pi (1+1+4+0+9+4) = 38\pi$。

【知识点深度剖析与扩展】

- 二级结论（定理应用）：
    

- 初值定理： $x[0] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\Omega}) d\Omega$。积分值等于 $2\pi$ 倍的 $k=0$ 处样值。
    
- 帕塞瓦尔定理 (Parseval's Theorem)： $\sum |x[k]|^2 = \frac{1}{2\pi} \int_{-\pi}^{\pi} |X(e^{j\Omega})|^2 d\Omega$。能量在时域和频域是守恒的。
    

- 物理意义： $X(e^{j0})$ 是信号的直流分量。$X(e^{j\pi})$ 是信号的最高频分量（奈奎斯特频率处的幅度）。
    

### 习题 1-14

【题目】

计算下列频谱函数对应的离散序列 $x[k]$：

(1) $X_1(e^{j\Omega}) = \sum_{n=-\infty}^{\infty} \delta(\Omega+2\pi n)$

(2) $X_2(e^{j\Omega}) = \frac{1-e^{j\Omega(N+1)}}{1-e^{-j\Omega}}$

(3) $X_3(e^{j\Omega}) = 1 + 2\sum_{n=1}^N \cos(\Omega n)$

(4) $X_4(e^{j\Omega}) = \frac{\alpha e^{j\Omega}}{(1-\alpha e^{-j\Omega})^2}, \quad |\alpha|<1$

【详细解答】

(1) 频域冲激串，对应时域常数。

$x_1[k] = \frac{1}{2\pi} \int_{-\pi}^{\pi} \delta(\Omega) e^{j\Omega k} d\Omega = \frac{1}{2\pi}$。

(2) 逆用等比求和 $\sum_{m=0}^{N} q^m = \frac{1-q^{N+1}}{1-q}$。

将分子写成 $1-e^{j\Omega(N+1)}$，分母乘 $e^{j\Omega}$ 整理：

$X_2 \approx -e^{j\Omega} \frac{1-e^{j\Omega(N+1)}}{1-e^{j\Omega}} = - \sum_{k=0}^{N} e^{j\Omega(k+1)}$。

对应频域是超前的指数和。

$x_2[k] = -1$，当 $-(N+1) \le k \le -1$。

(3) 利用欧拉公式：$2\cos(\Omega n) = e^{j\Omega n} + e^{-j\Omega n}$。

$X_3 = 1 + \sum_{n=1}^N e^{j\Omega n} + \sum_{n=1}^N e^{-j\Omega n} = \sum_{n=-N}^{N} e^{-j\Omega n}$。

这是矩形序列的 DTFT。

$x_3[k] = 1$，当 $|k| \le N$。

(4) 形式类似 $(n+1)a^n u[n]$ 的变换 $\frac{1}{(1-ae^{-j\Omega})^2}$。

这里是 $\frac{\alpha e^{j\Omega}}{(1-\alpha e^{-j\Omega})^2} = \alpha e^{j\Omega} \cdot \text{DTFT}\{(n+1)\alpha^n u[n]\}$。

利用时移特性（$e^{j\Omega}$ 对应 $k+1$）：

$x_4[k] = \alpha (k+1+1)\alpha^{k+1} u[k+1] = (k+2)\alpha^{k+2} u[k+1]$。

(注：若原题意图是考察 $k\alpha^k u[k]$，则应为 $\frac{\alpha e^{-j\Omega}}{(1-\alpha e^{-j\Omega})^2}$)

【知识点深度剖析与扩展】

- 常用变换对：
    

- 常数 $1 \leftrightarrow 2\pi \sum \delta(\Omega - 2\pi k)$。注意系数 $2\pi$。
    
- 对称方波（中心在0）：$x[k]=1, |k|\le N \leftrightarrow \frac{\sin((2N+1)\Omega/2)}{\sin(\Omega/2)}$。
    

- 技巧： 遇到余弦求和形式，优先考虑转换为指数求和，利用几何级数还原为矩形窗。
    

### 习题 1-15

【题目】

已知 $x[k]=R_7[k+3]$，$\tilde{Y}[m] = X(e^{j\Omega})|_{\Omega=2\pi m/3}$。不做 IDFS，求 $\tilde{y}[k]$。

【详细解答】

解析序列： $x[k]$ 是长度为 7 的矩形序列，范围 $-3 \le k \le 3$。

频域抽样定理： 频域抽样 $N=3$ 点，对应时域以 $N=3$ 为周期进行延拓（混叠）。

$\tilde{y}[k] = \sum_{r=-\infty}^{\infty} x[k+3r]$。

计算混叠： $x[k]$ 在主值区间 $[-3, 3]$ 的值为 $\{1, 1, 1, 1, 1, 1, 1\}$。

- $k=0$: $x[0] + x[3] + x[-3] = 1 + 1 + 1 = 3$。
    
- $k=1$: $x[1] + x[4] (\text{无}) + x[-2] = 1 + 0 + 1 = 2$。
    
- $k=2$: $x[2] + x[5] (\text{无}) + x[-1] = 1 + 0 + 1 = 2$。
    

结果：$\tilde{y}[k] = \{3, 2, 2\}$ (周期为3)。

【知识点深度剖析与扩展】

- 核心定理： 频域抽样 $\Leftrightarrow$ 时域周期延拓 (Time Aliasing)。$\tilde{x}[k] = \sum_{r=-\infty}^{\infty} x[k+rN]$。
    
- 应用： 这是理解 DFT 的关键。如果 $N$ 小于序列长度 $L$，则时域发生混叠，无法恢复。如果 $N \ge L$，则不混叠，可以通过补零后的 DFT 来代表原序列频谱。
    

### 习题 1-16

【题目】

已知有限长序列 $x[k]=\{1, 2, -1, 4, 3\}$。求该序列频谱 $X(e^{j\Omega})$ 在 $\Omega_m = 2\pi m/4, m=0,1,2,3$ 上的值。

【详细解答】

这等同于求 $x[k]$ 的 4 点 DFT。但序列长度 $L=5 > N=4$，需要先进行时域混叠（周期化）。

构造 4 点序列 $\tilde{x}_4[k] = \sum_r x[k+4r]$：

- $k=0$: $x[0] + x[4] = 1 + 3 = 4$。
    
- $k=1$: $x[1] = 2$。
    
- $k=2$: $x[2] = -1$。
    
- $k=3$: $x[3] = 4$。
    

计算 $\{4, 2, -1, 4\}$ 的 4 点 DFT：

- $X[0] = 4+2-1+4 = 9$。
    
- $X[1] = 4 + 2(-j) - 1(-1) + 4(j) = 5 + 2j$。
    
- $X[2] = 4 - 2 - 1 - 4 = -3$。
    
- $X[3] = 4 + 2(j) - 1(-1) + 4(-j) = 5 - 2j$。
    

【知识点深度剖析与扩展】

- 二级结论（长序列的 N 点 DFT）： 等于将原序列以 $N$ 为周期进行时域混叠后主值序列的 DFT。$X_N[k] = \text{DFT}_N \{ \sum_{r} x[n+rN] \}$。
    
- 应用： 这解释了为什么 DFT 的频率分辨率受限于 $N$。如果 $N$ 太小，时域混叠严重，频谱信息丢失。
    

### 习题 1-17

【题目】

已知 $x[k]=\alpha^k u[k], |\alpha|<1$。

(1) 求 $X(e^{j\Omega})$。

(2) 定义 $\tilde{x}[k] = \sum x[k+nN]$，求 $\tilde{x}[k]$ 及其 DFS $\tilde{X}[m]$。

(3) 给出 $\tilde{X}[m]$ 与 $X(e^{j\Omega})$ 的关系。

【详细解答】

(1) $X(e^{j\Omega}) = \frac{1}{1-\alpha e^{-j\Omega}}$。

(2) $\tilde{x}[k]$ 是 $x[k]$ 的周期化。在 $0 \le k < N$ 主值区间内：

$\tilde{x}[k] = \sum_{n=0}^{\infty} \alpha^{k+nN} = \alpha^k \sum_{n=0}^{\infty} (\alpha^N)^n = \frac{\alpha^k}{1-\alpha^N}$。

DFS $\tilde{X}[m]$： 由频域抽样定理，$\tilde{X}[m] = X(e^{j\Omega})|_{\Omega=2\pi m/N} = \frac{1}{1-\alpha e^{-j\frac{2\pi}{N}m}}$。

(3) 关系：$\tilde{X}[m]$ 是 $X(e^{j\Omega})$ 在单位圆上的等间隔采样。

【知识点深度剖析与扩展】

- 核心概念（DFS 与 DTFT 的关系）： 周期序列的 DFS 系数 $\tilde{X}[m]$ 正好等于其单周期主值序列的 DTFT 在 $\omega_k = \frac{2\pi}{N}k$ 处的采样值。
    
- 其他方案： 可以直接计算 $\tilde{x}[k]$ 的 DFS：$\sum_{k=0}^{N-1} \frac{\alpha^k}{1-\alpha^N} W_N^{mk} = \frac{1}{1-\alpha^N} \frac{1-(\alpha W_N^m)^N}{1-\alpha W_N^m}$。利用 $W_N^{mN}=1$，分子消去，得 $\frac{1}{1-\alpha W_N^m}$，结果一致。
    

### 习题 1-18

【题目】

已知 $H(e^{j\Omega}) = \begin{cases} e^{-j\Omega}, & |\Omega| \le \pi/4 \text{ 或 } |\Omega| \ge 3\pi/4 \\ 0, & \text{其他} \end{cases}$。求 $h[k]$。

【详细解答】

这是一个带阻滤波器。

$H(e^{j\Omega}) = [H_{LP}(\Omega, \frac{\pi}{4}) + H_{HP}(\Omega, \frac{3\pi}{4})] \cdot e^{-j\Omega}$。

利用理想低通响应 $h_{LP}[k] = \frac{\sin(\omega_c k)}{\pi k}$。

高通 $H_{HP} = 1 - H_{LP}$。

考虑相位因子 $e^{-j\Omega}$ 对应时移 $k \to k-1$。

- 低通部分（截止 $\pi/4$）：$\frac{\sin(\frac{\pi}{4}(k-1))}{\pi(k-1)}$。
    
- 高通部分（截止 $3\pi/4$）：$\delta[k-1] - \frac{\sin(\frac{3\pi}{4}(k-1))}{\pi(k-1)}$。
    

相加得：

$h[k] = \delta[k-1] - \frac{\sin(0.75\pi(k-1))}{\pi(k-1)} + \frac{\sin(0.25\pi(k-1))}{\pi(k-1)}$。

【知识点深度剖析与扩展】

- 二级结论（滤波器关系）：
    

- $h_{HP}[k] = (-1)^k h_{LP, \pi-\omega_c}[k]$ (频谱搬移法)。
    
- $h_{HP}[k] = \delta[k] - h_{LP, \omega_c}[k]$ (全通减低通法)。
    
- $h_{BS}[k] = \delta[k] - h_{BP}[k]$。
    

- 线性相位： 因子 $e^{-j\alpha \Omega}$ 对应时域延迟 $\alpha$。对于理想滤波器，这保证了群时延为常数，波形不失真。
    

### 习题 1-19

【题目】

$H(e^{j\Omega}) = e^{-j3\Omega}, |\Omega| < 3\pi/16$ (其余为0)。输入 $x[k] = \sum \delta[k-16n]$。求输出 $y[k]$。

【详细解答】

(1) $h[k]$：理想低通加时移。$h[k] = \frac{\sin(\frac{3\pi}{16}(k-3))}{\pi(k-3)}$。

(2) 输入是周期 $N=16$ 的脉冲串。频谱是离散谐波，基波频率 $\Omega_0 = 2\pi/16 = \pi/8$。

谐波位置：$0, \pm \pi/8, \pm 2\pi/8 \dots$

滤波器截止频率 $\omega_c = 3\pi/16 = 1.5 (\pi/8)$。

能通过的谐波只有：$0$ (直流), $\pi/8$ (基波), $-\pi/8$。

$Y(e^{j\Omega}) = H(e^{j0}) \cdot 2\pi\delta(\Omega) + H(e^{j\pi/8}) \cdot 2\pi\delta(\Omega-\pi/8) + \dots$

利用特征函数法：

- 直流分量：$1 \cdot \frac{1}{16}$ (DFS系数)。
    
- 基波分量：$e^{-j3(\pi/8)} \cdot \frac{1}{16} e^{j\frac{\pi}{8}k} + e^{-j3(-\pi/8)} \cdot \frac{1}{16} e^{-j\frac{\pi}{8}k}$。
    

$y[k] = \frac{1}{16} [ 1 + 2\cos(\frac{\pi}{8}k - \frac{3\pi}{8}) ] = \frac{1}{16} [ 1 + 2\cos(\frac{\pi}{8}(k-3)) ]$。

【知识点深度剖析与扩展】

- 二级结论（周期冲激串的频谱）： $\sum_k \delta[k-nN] \leftrightarrow \frac{2\pi}{N} \sum_k \delta(\Omega - \frac{2\pi}{N}k)$。时域周期脉冲，频域也是周期脉冲。
    
- 特征函数法 (Eigenfunction Property)： $e^{j\Omega_0 k} \to H(e^{j\Omega_0})e^{j\Omega_0 k}$。对于周期信号，先展开为傅里叶级数，再对每一项分别乘频率响应。
    

### 习题 1-20

【题目】

$H(e^{j\Omega}) = e^{-j2\Omega}, 0.3\pi < |\Omega| < 0.6\pi$。

输入 $x[k] = \cos(0.2\pi k) + 0.8\sin(0.5\pi k) + 0.6\sin(0.7\pi k)$。求 $y[k]$。

(1) $h[k]$：带通滤波器。$h[k] = \frac{\sin(0.6\pi(k-2))}{\pi(k-2)} - \frac{\sin(0.3\pi(k-2))}{\pi(k-2)}$。

【知识点深度剖析与扩展】

- 考点： 滤波器的选频特性。
    
- 分析： 题目考查是否能快速判断频率成分是否落在通带内。注意相位滞后对输出波形的影响（$\sin(\theta-\pi) = -\sin\theta$）。
    

## 【第三板块：$z$ 变换与系统分析】

### 习题 1-21

【题目】

利用 $z$ 变换计算 $x[k] = \alpha^k u[k] * \beta^k u[k]$。

【详细解答】

$X(z) = \frac{1}{1-\alpha z^{-1}}, \quad H(z) = \frac{1}{1-\beta z^{-1}}$。

$Y(z) = X(z)H(z) = \frac{1}{(1-\alpha z^{-1})(1-\beta z^{-1})}$。

假设 $\alpha \neq \beta$，部分分式展开：

$Y(z) = \frac{A}{1-\alpha z^{-1}} + \frac{B}{1-\beta z^{-1}}$。

求系数：$A = \frac{1}{1-\beta/\alpha} = \frac{\alpha}{\alpha-\beta}, \quad B = \frac{1}{1-\alpha/\beta} = \frac{\beta}{\beta-\alpha}$。

$Y(z) = \frac{\alpha}{\alpha-\beta} \frac{1}{1-\alpha z^{-1}} - \frac{\beta}{\alpha-\beta} \frac{1}{1-\beta z^{-1}}$。

反变换（ROC $|z|>\max(|\alpha|,|\beta|)$）：

$y[k] = [\frac{\alpha}{\alpha-\beta} \alpha^k - \frac{\beta}{\alpha-\beta} \beta^k] u[k] = \frac{\alpha^{k+1}-\beta^{k+1}}{\alpha-\beta} u[k]$。

【知识点深度剖析与扩展】

- 二级结论（留数法）： 部分分式系数 $A_k = Y(z)(1-p_k z^{-1})|_{z=p_k}$。
    
- 其他方案：
    

- 时域直接卷积：如习题1-1。
    
- DTFT法：原理相同，但在频域操作不如Z域代数运算方便。
    

### 习题 1-22

【题目】

已知 $X(z) = \frac{2-0.5z^{-1}}{(1+z^{-1})(1-1.5z^{-1})}$，求不同收敛域对应的 $x[k]$。

【详细解答】

极点：$p_1 = -1, p_2 = 1.5$。

部分分式展开：$X(z) = \frac{1}{1+z^{-1}} + \frac{1}{1-1.5z^{-1}}$。

1. $|z|>1.5$：两个极点都在圆内，均为右边序列。  
    $x[k] = (-1)^k u[k] + (1.5)^k u[k]$。
    
2. $1<|z|<1.5$：$-1$ 对应右边，$1.5$ 对应左边。  
    $x[k] = (-1)^k u[k] - (1.5)^k u[-k-1]$。
    
3. $|z|<1$：两个极点都在圆外，均为左边序列。  
    $x[k] = -(-1)^k u[-k-1] - (1.5)^k u[-k-1]$。
    

【知识点深度剖析与扩展】

- 核心规律：
    

- ROC 在极点外侧 $\rightarrow$ 因果/右边序列 ($u[k]$)。
    
- ROC 在极点内侧 $\rightarrow$ 反因果/左边序列 ($-u[-k-1]$)。
    

- 应用： 这是判断系统因果性和稳定性的依据。若要求系统稳定（包含单位圆），则根据极点模值分布唯一确定序列形式。
    

### 习题 1-23

【题目】

实序列 $x[k]$ 的自相关 $r_x[n]$。

(1) 确定 $R_x(z)$ 及 ROC。

(2) 设 $x[k]=a^k u[k], |a|<1$，求 $R_x(z)$ 及 $r_x[n]$。

【详细解答】

(1) $r_x[n] = x[n] * x[-n]$。

$\mathcal{Z}\{x[-n]\} = X(z^{-1})$，ROC 为 $1/R_{x+}$ 到 $1/R_{x-}$。

$R_x(z) = X(z)X(z^{-1})$。

ROC 为 $X(z)$ 与 $X(z^{-1})$ ROC 的交集。

(2) $X(z) = \frac{1}{1-az^{-1}}, |z|>|a|$。

$X(z^{-1}) = \frac{1}{1-az}, |z|<1/|a|$。

$R_x(z) = \frac{1}{(1-az^{-1})(1-az)} = \frac{-a^{-1}z}{(z-a)(z-a^{-1})}$。

ROC: $|a| < |z| < 1/|a|$。

利用部分分式和双边序列反变换可得 $r_x[n] = \frac{1}{1-a^2} a^{|n|}$。

【知识点深度剖析与扩展】

- 二级结论（功率谱密度 PSD）： $S_x(e^{j\Omega}) = R_x(z)|_{z=e^{j\Omega}} = |X(e^{j\Omega})|^2$。
    
- 谱分解定理： 功率谱可以分解为最小相位系统 $H_{min}(z)$ 与其共轭倒数系统的乘积，这在维纳滤波中非常重要。
    
- 性质： $R_x(z)$ 的零极点关于单位圆镜像对称（$p \leftrightarrow 1/p$）。
    

### 习题 1-24

【题目】

已知稳定系统 $H(z) = \frac{2(1-z^{-1})}{(1-0.5z^{-1})(1-2z^{-1})}$，求 $h[k]$。

【详细解答】

极点：$p_1=0.5, p_2=2$。

系统稳定 $\Rightarrow$ ROC 必须包含单位圆 ($|z|=1$)。所以 ROC 必须选在环状区域：$0.5 < |z| < 2$。

部分分式展开：$H(z) = \frac{2/3}{1-0.5z^{-1}} + \frac{4/3}{1-2z^{-1}}$。

因为 ROC $0.5 < |z| < 2$：

- $0.5$ 极点对应右边序列（因果）。
    
- $2$ 极点对应左边序列（反因果）。
    

$h[k] = \frac{2}{3}(0.5)^k u[k] - \frac{4}{3}(2)^k u[-k-1]$。

【知识点深度剖析与扩展】

- 核心考点（稳定性与 ROC 的关系）：
    

- 稳定 $\Leftrightarrow$ ROC 包含单位圆。
    
- 因果 $\Leftrightarrow$ ROC 为 $|z| > R_{max}$。
    
- 稳定且因果 $\Leftrightarrow$ 所有极点都在单位圆内。
    

- 分析： 本题系统稳定但非因果，说明它是一个双边系统。
    

### 习题 1-25

【题目】

系统如图所示（前馈结构）：$y[k] = x[k] + 2x[k-2] + x[k-4]$。

(1) 求 $h[k]$。

(2) 输入 $x[k]=\cos(\pi k)$，求响应。

【详细解答】

(1) 根据差分方程直接写出 $h[k]$：$h[k] = \delta[k] + 2\delta[k-2] + \delta[k-4]$。

(2) 求频率响应：

$H(e^{j\Omega}) = 1 + 2e^{-j2\Omega} + e^{-j4\Omega} = e^{-j2\Omega}(e^{j2\Omega} + 2 + e^{-j2\Omega}) = e^{-j2\Omega}(2 + 2\cos(2\Omega))$。

输入频率 $\Omega = \pi$。

$H(e^{j\pi}) = e^{-j2\pi}(2 + 2\cos(2\pi)) = 1 \cdot (2+2) = 4$。

稳态响应 $y[k] = |H| \cos(\pi k + \angle H) = 4 \cos(\pi k)$。

【知识点深度剖析与扩展】

- 二级结论（线性相位 FIR）： 单位脉冲响应对称 ($h[k]=h[N-1-k]$) 的 FIR 系统具有线性相位。本题 $h=\{1, 0, 2, 0, 1\}$ 偶对称，相位为 $-2\Omega$ (线性)。
    
- 其他方案： Z变换法：$Y(z) = (1+2z^{-2}+z^{-4})X(z)$。直接卷积计算。
    

### 习题 1-26

【题目】

设 $H(z)$ 为理想低通（截止频率 $\pi/2$），画出下列系统的幅度响应：

(1) $H_1(z)=H(-z)$

(2) $H_2(z)=H(z^2)$

(3) $H_3(z)=H(z)H(z^2)$

【详细解答】

(1) $z \to -z \Rightarrow e^{j\Omega} \to e^{j(\Omega-\pi)}$。

频谱搬移 $\pi$。原 $[-\pi/2, \pi/2]$ 的通带搬移到 $[ \pi/2, 3\pi/2 ]$。

结果： 变成高通滤波器。

(2) $z \to z^2 \Rightarrow \Omega \to 2\Omega$。

频谱压缩 2 倍。原 $[-\pi/2, \pi/2]$ 压缩为 $[-\pi/4, \pi/4]$。同时，由于周期性，在 $\pm \pi$ 附近会出现镜像通带。

结果： 变成多带滤波器。

(3) 级联。$H(z)$ 通带 $[-\pi/2, \pi/2]$ 与 $H(z^2)$ 通带（$[-\pi/4, \pi/4]$ 及高频部分）取交集。

结果： 是更窄的低通滤波器，截止频率 $\pi/4$。

【知识点深度剖析与扩展】

- 二级结论（多抽样率处理）：
    

- $H(z^M)$：对应内插 (Interpolation) 中的镜像滤波器。频谱压缩 $M$ 倍，产生 $M-1$ 个镜像。
    
- $H(z)H(z^2)H(z^4)\dots$：这种级联结构常用于实现窄带滤波器或抽取滤波器。
    

- 变换规律：
    

- $z \to -z$：低通变高通。
    
- $z \to 1/z$：时反（幅度响应不变）。
    

### 习题 1-27

【题目】

$H(z) = \frac{0.5-z^{-1}}{1-0.6z^{-1}}$。求幅度响应相同的最小相位系统 $H_{min}(z)$。

【详细解答】

极点 $0.6$ 在圆内，稳定。

零点 $z_0 = 1/0.5 = 2$ 在圆外，非最小相位。

方法：利用全通滤波器翻转零点。

全通因子 $A(z) = \frac{0.5-z^{-1}}{1-0.5z^{-1}}$。

$H(z) = \frac{1-0.5z^{-1}}{1-0.6z^{-1}} \cdot \frac{0.5-z^{-1}}{1-0.5z^{-1}} = H_{min}(z) \cdot A(z)$。

所以 $H_{min}(z) = \frac{1-0.5z^{-1}}{1-0.6z^{-1}}$。

【知识点深度剖析与扩展】

- 核心概念：
    

- 最小相位系统：所有零极点均在单位圆内。具有最小群时延和最小能量延迟。
    
- 全通系统：幅度恒为 1 ($|H(e^{j\Omega})|=1$)。极点 $p_k$ 与零点 $1/p_k^*$ 成对出现。
    

- 通用方法： 将圆外的零点 $(z^{-1}-z_0^*)$ 替换为圆内的零点 $(1-z_0 z^{-1})$，并注意增益调整。
    

### 习题 1-28

【题目】

某系统极点为 $\pm 1/3$，零点为 2。$H(1)=1$。

(1) 求 $H(z)$。

(2) 求 $H_{min}(z)$。

【详细解答】

(1) $H(z) = K \frac{z-2}{(z-1/3)(z+1/3)}$。

代入 $H(1)=1$: $K \frac{1-2}{(1-1/3)(1+1/3)} = K \frac{-1}{8/9} = 1 \Rightarrow K = -8/9$。

$H(z) = -\frac{8}{9} \frac{z-2}{z^2-1/9}$。

(2) 将圆外零点 $2$ 映射到圆内 $1/2=0.5$。

构造 $H_{min}(z)$ 形式：$K' \frac{z-0.5}{z^2-1/9}$。

为了保持幅度响应 $|H(e^{j\Omega})|$ 不变，利用全通因子翻转：

$H(z) = H_{min}(z) \frac{z-2}{1-2z} \Rightarrow H_{min}(z) = H(z) \frac{1-2z}{z-2} = -\frac{8}{9} \frac{1-2z}{z^2-1/9}$。

【知识点深度剖析与扩展】

- 零极点图分析： 极点靠近单位圆，频响峰值越尖锐。零点靠近单位圆，频响谷值越深（陷波）。
    
- 分析： 通过零极点图可以直接定性画出幅度响应草图，这是 DSP 直觉的重要来源。
    

### 习题 1-29

【题目】

将 $H(z) = \frac{(3+z^{-1})(0.2+z^{-1})}{(1+0.4z^{-1})(1+0.6z^{-1})}$ 分解为最小相位和全通级联。

【详细解答】

检查零点：

- $3+z^{-1}=0 \Rightarrow z = -1/3$ (圆内，最小相位)。
    
- $0.2+z^{-1}=0 \Rightarrow z = -5$ (圆外，非最小相位)。
    

需要处理因子 $(0.2+z^{-1})$。构造全通滤波器，消去该因子并引入共轭倒数零点。

$H(z) = \frac{3+z^{-1}}{D(z)} \cdot (1+0.2z^{-1}) \cdot \frac{0.2+z^{-1}}{1+0.2z^{-1}}$。

$H_{min}(z) = \frac{(3+z^{-1})(1+0.2z^{-1})}{(1+0.4z^{-1})(1+0.6z^{-1})}$。

$A(z) = \frac{0.2+z^{-1}}{1+0.2z^{-1}}$。

【知识点深度剖析与扩展】

- 分解定理： 任意有理系统函数可分解为 $H(z) = H_{min}(z) H_{ap}(z)$。
    
- 应用： 在信道均衡中，通常先通过全通滤波器校正相位，再通过最小相位逆滤波器校正幅度，因为最小相位系统的逆系统也是稳定的。
    

## 【第四板块：信号抽样与重建】

### 习题 1-30

【题目】

处理框图：抽样 $\to$ 离散处理 $\to$ 重建。$T=0.01$。$x(t)$ 频谱为三角波，截止频率 $50\pi$。求各点频谱。

【详细解答】

抽样频率： $\Omega_s = 2\pi/T = 200\pi$。

信号最高频： $\Omega_m = 50\pi$。

混叠判断： $200\pi > 2 \times 50\pi$，无混叠。

离散频谱 $X(e^{j\Omega})$： 是 $X(j\omega)$ 的周期延拓，幅度乘以 $1/T$。数字频率 $\Omega = \omega T$。

原截止 $50\pi$ 对应数字截止 $\Omega_c = 50\pi \times 0.01 = 0.5\pi$。

所以 $X(e^{j\Omega})$ 是周期为 $2\pi$ 的三角波，基带宽度 $[-0.5\pi, 0.5\pi]$。

重建： 无混叠，理想重建，恢复原信号。

【知识点深度剖析与扩展】

- 核心公式： $X(e^{j\Omega}) = \frac{1}{T} \sum_{k} X_a(j\frac{\Omega-2\pi k}{T})$。
    
- 数字频率与模拟频率： $\Omega = \omega T$。这是连接两个世界的桥梁。
    
- 泊松求和公式 (Poisson Summation Formula)： 上述公式的数学基础，揭示了时域离散化导致频域周期化。
    

### 习题 1-31

【题目】

同上，但 $T=0.02$。

【详细解答】

抽样频率： $\Omega_s = 100\pi$。

临界判断： $100\pi = 2 \times 50\pi$。刚好临界采样。

频谱： 周期延拓时，频谱边缘刚好相切，不重叠。

数字截止频率 $\Omega_c = 50\pi \times 0.02 = \pi$。

$X(e^{j\Omega})$ 充满了整个 $[-\pi, \pi]$ 区间。

重建： 仍能恢复。但如果 $T>0.02$ 则发生混叠。

【知识点深度剖析与扩展】

- 奈奎斯特定理： $f_s \ge 2 f_{max}$。注意是大于等于。
    
- 工程实践： 实际中滤波器不是理想的，需要保护带，因此采样率通常略高于 $2 f_{max}$（如音频 44.1kHz 针对 20kHz）。
    

### 习题 1-32

【题目】

同上，$T=0.001$。处理过程：$x[k]$ 乘以 $(-1)^k$。求输出 $y(t)$。

【详细解答】

抽样： $\Omega_s = 2000\pi$。$\Omega_m = 50\pi$。严重过采样。

数字截止 $\Omega_c = 50\pi \times 0.001 = 0.05\pi$。

处理： $w[k] = x[k](-1)^k = x[k] e^{j\pi k}$。频域搬移 $\pi$。

$W(e^{j\Omega})$ 的中心从 $0$ 移到了 $\pi$。

重建： 数字频率 $\pi$ 对应模拟频率 $\pi/T = 1000\pi = \Omega_s/2$。

原低频信号变成了中心在 $1000\pi$ 的高频窄带信号（带通信号）。

$y(t)$ 是 $x(t)$ 调制在 $1000\pi$ 载波上的结果。

【知识点深度剖析与扩展】

- 频谱倒置 (Spectral Inversion)： $(-1)^k$ 相当于把低频搬到高频，高频搬到低频。在模拟域对应搬移 $\omega_s/2$。
    
- 应用： 这是一种简单的数字混频技术。也可以用来把高通滤波器转换为低通滤波器，反之亦然。
