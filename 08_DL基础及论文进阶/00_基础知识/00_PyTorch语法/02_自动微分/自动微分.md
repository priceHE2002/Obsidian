# 2. 自动微分

PyTorch 的 `autograd` 是自动求导引擎，它记录所有张量操作并构建计算图，在调用 `.backward()` 时自动计算梯度。这就是反向传播的工程实现。

## 2.1 基础使用

```python
import torch

# requires_grad=True 表示需要跟踪梯度
x = torch.tensor([2.0], requires_grad=True)
w = torch.tensor([0.5], requires_grad=True)
b = torch.tensor([0.1], requires_grad=True)

# 前向计算（autograd 自动记录计算图）
y = w * x + b

# 反向传播：计算 dy/dw, dy/dx, dy/db
y.backward()

print(x.grad)  # tensor([0.5000]) — dy/dx = w
print(w.grad)  # tensor([2.0000]) — dy/dw = x
print(b.grad)  # tensor([1.0000]) — dy/db = 1
```

## 2.2 标量与非标量反向传播

```python
# 情况一：损失是标量 → 直接 backward
loss = (y - 3.0) ** 2   # loss 是标量
loss.backward()

# 情况二：输出是向量 → 需要传入 gradient 参数
x = torch.randn(3, requires_grad=True)
y = x * 2                # y 是向量 [3]

# 如果每个输出元素对 x 的梯度同等重要
y.backward(torch.ones_like(y))
print(x.grad)            # 全为 2
```

核心理解：`backward()` 本质是计算 $\frac{\partial L}{\partial x}$。如果 `y` 已经是标量（loss），无需传参；如果是向量，需要用一个 shape 相同的张量表示 $L$ 对各元素的初始梯度。

## 2.3 梯度累积与清零

```python
w = torch.tensor([1.0], requires_grad=True)

for i in range(3):
    loss = (w - 2) ** 2
    loss.backward()
    print(f"Step {i}: grad = {w.grad}")
    # 注意：梯度会累积！每次 backward 是累加，不会自动清零

# 正确写法
for i in range(3):
    if w.grad is not None:
        w.grad.zero_()     # 手动清零
    loss = (w - 2) ** 2
    loss.backward()
    with torch.no_grad():   # 更新时不记录计算图
        w -= 0.1 * w.grad
```

## 2.4 禁用梯度跟踪

```python
# 方式一：torch.no_grad() — 推理 / 评估时使用
with torch.no_grad():
    y = model(x)  # 不构建计算图，省显存，更快

# 方式二：detach() — 将张量从计算图中剥离
x = torch.randn(3, requires_grad=True)
y = x * 2
y_detached = y.detach()  # y_detached 不再与 x 有梯度关系

# 方式三：requires_grad_(False) — 原地关闭
x.requires_grad_(False)
```

## 2.5 计算图保留

```python
x = torch.tensor([2.0], requires_grad=True)
y = x ** 2

# 默认行为：backward 后释放计算图（节省内存）
y.backward()
# y.backward()  # 报错！计算图已释放

# 如果需要多次 backward（不常见）
x = torch.tensor([2.0], requires_grad=True)
y = x ** 2
y.backward(retain_graph=True)  # 保留计算图
y.backward()                    # 第二次，正常
```

## 2.6 高阶梯度

```python
x = torch.tensor([2.0], requires_grad=True)
y = x ** 3

# 一阶梯度
grad1 = torch.autograd.grad(y, x, create_graph=True)[0]
print(grad1)  # 3*x^2 = 12

# 二阶梯度（对一阶梯度再求导）
grad2 = torch.autograd.grad(grad1, x)[0]
print(grad2)  # 6*x = 12
```
