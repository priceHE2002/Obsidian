# 4. 数据加载

PyTorch 的数据加载由 `Dataset` 和 `DataLoader` 配合完成。`Dataset` 负责从磁盘读取并预处理单个样本，`DataLoader` 负责批量组装、打乱和多进程加速。

## 4.1 Dataset 基类

```python
from torch.utils.data import Dataset

class MyDataset(Dataset):
    def __init__(self, data, labels, transform=None):
        self.data = data
        self.labels = labels
        self.transform = transform

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        x = self.data[idx]
        y = self.labels[idx]

        if self.transform:
            x = self.transform(x)

        return x, y
```

**三个必须实现的方法**：

- `__len__`：返回数据集大小。
- `__getitem__`：根据索引返回一个 `(data, label)` 样本。
- `__init__`：初始化，指定文件路径或数据源。

## 4.2 DataLoader 用法

```python
from torch.utils.data import DataLoader

dataset = MyDataset(data, labels)
dataloader = DataLoader(
    dataset,
    batch_size=32,       # 每个 batch 多少样本
    shuffle=True,        # 每个 epoch 是否打乱
    num_workers=4,       # 多进程加载（0 表示主进程）
    pin_memory=True,     # 数据预加载到 GPU 页锁定内存（训练快 10-20%）
    drop_last=True,      # 丢弃最后不足一个 batch 的数据
)

for batch_idx, (x, y) in enumerate(dataloader):
    x, y = x.to(device), y.to(device)
    # 训练逻辑...
```

## 4.3 torchvision 内置数据集

```python
from torchvision import datasets, transforms

# 图像预处理
transform = transforms.Compose([
    transforms.Resize((32, 32)),
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.5, 0.5, 0.5],
                         std=[0.5, 0.5, 0.5])
])

# CIFAR-10 示例（自动下载）
train_dataset = datasets.CIFAR10(
    root='./data', train=True, download=True,
    transform=transform
)

test_dataset = datasets.CIFAR10(
    root='./data', train=False, download=True,
    transform=transform
)
```

常用内置数据集：`MNIST`、`FashionMNIST`、`CIFAR10`、`CIFAR100`、`ImageNet`（需手动下载）、`COCO`。

## 4.4 常用预处理变换

```python
from torchvision import transforms

train_transform = transforms.Compose([
    transforms.RandomCrop(32, padding=4),          # 随机裁剪
    transforms.RandomHorizontalFlip(p=0.5),         # 随机水平翻转
    transforms.ColorJitter(brightness=0.2,          # 颜色抖动
                           contrast=0.2,
                           saturation=0.2),
    transforms.ToTensor(),                          # PIL Image → Tensor [0,1]
    transforms.Normalize(mean=[0.485, 0.456, 0.406],
                         std=[0.229, 0.224, 0.225]) # 标准化
])

# 验证集只做 ToTensor + Normalize（不做增强）
val_transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize(mean=[0.485, 0.456, 0.406],
                         std=[0.229, 0.224, 0.225])
])
```

**关键原则**：训练集用数据增强 + 标准化；验证/测试集只用标准化。Normalize 使用的均值和标准差通常来自 ImageNet 统计值，或从自己的训练集计算。

## 4.5 自定义图像数据集

```python
import os
from PIL import Image

class ImageFolderDataset(Dataset):
    """从文件夹加载图像分类数据"""
    def __init__(self, root_dir, transform=None):
        self.root_dir = root_dir
        self.transform = transform
        self.classes = sorted(os.listdir(root_dir))

        self.samples = []
        for label, class_name in enumerate(self.classes):
            class_dir = os.path.join(root_dir, class_name)
            for fname in os.listdir(class_dir):
                if fname.endswith(('.jpg', '.png', '.jpeg')):
                    self.samples.append((
                        os.path.join(class_dir, fname), label
                    ))

    def __len__(self):
        return len(self.samples)

    def __getitem__(self, idx):
        path, label = self.samples[idx]
        img = Image.open(path).convert('RGB')
        if self.transform:
            img = self.transform(img)
        return img, label
```

> 实际上 `torchvision.datasets.ImageFolder` 已经实现了上述功能，一般直接用它即可。

```python

# 按比例划分
total = len(dataset)
train_size = int(0.8 * total)
val_size = total - train_size

    dataset, [train_size, val_size]
)
```
