# 6.6 架构组装对照

将前面各节的基础组件组装成完整的 Transformer Encoder Block、LLaMA Block 和多层 Encoder。

> **依赖说明**：本章使用的类均定义在前几节中。运行完整代码时需先导入 `6.2_基础层对照` 中的 `dropout_forward_np`、`6.4_注意力机制对照` 中的 `MultiHeadAttention_Numpy` 和 `GroupedQueryAttention_Numpy`、`6.5_Transformer层对照` 中的 `FeedForward_Numpy`、`SwiGLU_FFN_Numpy`、`LayerNorm_Numpy`、`RMSNorm_Numpy`。

## 6.6.1 原始 Transformer Encoder Block

```python
class TransformerEncoderBlock_Numpy:
    """
    torch.nn.TransformerEncoderLayer 的 NumPy 实现
    结构（Post-Norm）：
        x → MultiHeadAttention (自注意力) → +残差 → LayerNorm
          → FeedForward → +残差 → LayerNorm
    """

    def __init__(self, d_model=512, n_heads=8, d_ff=2048, dropout=0.1):
        self.self_attn = MultiHeadAttention_Numpy(d_model, n_heads)
        self.ffn = FeedForward_Numpy(d_model, d_ff)
        self.norm1 = LayerNorm_Numpy(d_model)
        self.norm2 = LayerNorm_Numpy(d_model)
        self.dropout = dropout

    def forward(self, x, mask=None, training=True):
        # 1. 自注意力 + 残差 + LayerNorm
        attn_out = self.self_attn.forward(x, x, x, mask=mask)
        if training:
            attn_out = dropout_forward_np(attn_out, p=self.dropout, training=True)
        x = self.norm1.forward(x + attn_out)

        # 2. FeedForward + 残差 + LayerNorm
        ffn_out = self.ffn.forward(x)
        if training:
            ffn_out = dropout_forward_np(ffn_out, p=self.dropout, training=True)
        x = self.norm2.forward(x + ffn_out)

        return x

    def backward(self, dout):
        # FFN + 残差 + LN2
        dffn = self.norm2.backward(dout)
        dx = dffn + dout  # 残差梯度
        dx = self.ffn.backward(dx)

        # Attention + 残差 + LN1
        dattn = self.norm1.backward(dx)
        dx2 = dattn + dx   # 残差梯度
        dx2 = self.self_attn.backward(dx2)

        return dx2

    def step(self, lr):
        self.self_attn.step(lr)
        self.ffn.step(lr)
        self.norm1.step(lr)
        self.norm2.step(lr)


# ── 测试 ──
encoder_block = TransformerEncoderBlock_Numpy(d_model=512, n_heads=8, d_ff=2048)
x = np.random.randn(4, 10, 512)

out = encoder_block.forward(x, training=True)
print("Encoder Block 输出形状:", out.shape)  # (4, 10, 512)

causal_mask = np.tril(np.ones((10, 10)))
out_causal = encoder_block.forward(x, mask=causal_mask, training=True)
print("因果 Encoder 输出形状:", out_causal.shape)
```

## 6.6.2 LLaMA 风格 Encoder Block

```python
class LLaMAEncoderBlock_Numpy:
    """
    LLaMA 风格的 Transformer Block（现代大模型标准配置）
    结构（Pre-Norm 变体）：
        x → RMSNorm → GQA (RoPE) → +残差
        x → RMSNorm → SwiGLU_FFN → +残差

    对比原始 Transformer：
        原始: LN → MHA → +残差 → LN → FFN → +残差
        LLaMA: RMSNorm → GQA+RoPE → +残差 → RMSNorm → SwiGLU → +残差
    """

    def __init__(self, d_model=512, n_query_heads=32, n_kv_heads=8,
                 d_ff=None, dropout=0.1):
        self.d_model = d_model
        self.d_ff = d_ff or int(8/3 * d_model)

        self.self_attn = GroupedQueryAttention_Numpy(
            d_model, n_query_heads, n_kv_heads
        )
        self.ffn = SwiGLU_FFN_Numpy(d_model, self.d_ff)
        self.norm1 = RMSNorm_Numpy(d_model)
        self.norm2 = RMSNorm_Numpy(d_model)
        self.dropout = dropout

    def forward(self, x, mask=None, training=True):
        # 1. Pre-Norm → GQA → +残差
        normed = self.norm1.forward(x)
        attn_out = self.self_attn.forward(normed, mask=mask)
        if training:
            attn_out = dropout_forward_np(attn_out, p=self.dropout, training=True)
        x = x + attn_out

        # 2. Pre-Norm → SwiGLU FFN → +残差
        normed = self.norm2.forward(x)
        ffn_out = self.ffn.forward(normed)
        if training:
            ffn_out = dropout_forward_np(ffn_out, p=self.dropout, training=True)
        x = x + ffn_out

        return x

    def backward(self, dout):
        # FFN 残差反向
        dffn_res = self.ffn.backward(dout)
        dnorm2 = self.norm2.backward(dffn_res)
        dx = dout + dnorm2

        # Attention 残差反向
        dattn_res = self.self_attn.backward(dx)
        dnorm1 = self.norm1.backward(dattn_res)
        dx2 = dx + dnorm1

        return dx2

    def step(self, lr):
        self.self_attn.step(lr)
        self.ffn.step(lr)
        self.norm1.step(lr)
        self.norm2.step(lr)


# ── 测试 ──
llama_block = LLaMAEncoderBlock_Numpy(
    d_model=512, n_query_heads=32, n_kv_heads=8
)
x = np.random.randn(2, 16, 512)
out = llama_block.forward(x, training=True)
print("LLaMA Block 输出形状:", out.shape)  # (2, 16, 512)

diff = np.linalg.norm(out[0, 0] - x[0, 0]) / np.linalg.norm(x[0, 0])
print(f"残差变化比例: {diff:.3f}")
```

## 6.6.3 原始 Transformer vs LLaMA 架构对比

| 组件 | 原始 Transformer (2017) | LLaMA 风格 (2023+) |
|---|---|---|
| 位置编码 | Sinusoidal PE（加到输入） | RoPE（注入 Q/K 的计算中） |
| 归一化 | LayerNorm（Post-Norm） | RMSNorm（Pre-Norm） |
| 注意力 | MHA（每个头独立 Q/K/V） | GQA（多 Q 头共享 K/V 头） |
| 前馈网络 | ReLU(xW1+b)W2+b | SwiGLU: (Swish(xWg)⊙(xWu))Wd |
| 激活函数 | ReLU（FFN 中） | SiLU/Swish（门控中） |
| 残差位置 | LayerNorm 在残差之后（Post-Norm） | RMSNorm 在子层之前（Pre-Norm） |

**Pre-Norm vs Post-Norm 的面试要点**：

- **Post-Norm**（原始）：`x + Sublayer(LN(x))` → LN 在残差加法之后，早期层的梯度需要通过多层 LN 传播，深层训练不稳定，需要 warmup。
- **Pre-Norm**（LLaMA）：`LN(x)` 在残差支路前 → 梯度可以通过残差连接更直接地向前传播，训练更稳定，对学习率更鲁棒。实际上 LLaMA 用 RMSNorm 替代 LN 后更是只需 `RMS(x)` 一次计算。

## 6.6.4 多层 Encoder 堆叠

```python
class TransformerEncoder_Numpy:
    """
    由多个 TransformerEncoderBlock 堆叠而成
    等价于 torch.nn.TransformerEncoder
    """

    def __init__(self, num_layers=6, d_model=512, n_heads=8, d_ff=2048, dropout=0.1):
        self.layers = [
            TransformerEncoderBlock_Numpy(d_model, n_heads, d_ff, dropout)
            for _ in range(num_layers)
        ]
        self.d_model = d_model

    def forward(self, x, mask=None, training=True):
        for layer in self.layers:
            x = layer.forward(x, mask=mask, training=training)
        return x

    def backward(self, dout):
        for layer in reversed(self.layers):
            dout = layer.backward(dout)
        return dout

    def step(self, lr):
        for layer in self.layers:
            layer.step(lr)


# ── 测试：6 层 encoder ──
encoder = TransformerEncoder_Numpy(num_layers=6, d_model=512, n_heads=8, d_ff=2048)
x = np.random.randn(4, 10, 512)
out = encoder.forward(x, training=True)
print("6 层 Encoder 输出形状:", out.shape)  # (4, 10, 512)
print("最后一层输出范数:", np.linalg.norm(out[0, 0]))
```

## 6.6.5 LN vs BN 对比

| 维度 | BatchNorm | LayerNorm |
|---|---|---|
| 归一化维度 | 同一通道，跨 N, H, W | 同一样本，跨 C |
| 对 batch size 依赖 | 强（小 batch 统计不稳定） | 无（每个样本独立计算） |
| 训练/推理行为 | 不同（推理用 running 统计） | 相同（每样本独立计算） |
| 常见用途 | CNN 分类/检测 | RNN、Transformer |
| 面试一句话 | "BN 归一化同一特征跨样本分布" | "LN 归一化同一样本跨特征分布" |

**为什么 Transformer 用 LN 而不用 BN？**

1. 序列长度可变，batch 内 padding 噪声干扰 BN 统计。
2. 自回归生成时 batch=1，BN 直接失效。
3. LN 训练推理行为一致，无需维护 running statistics。

## 6.6.6 DataLoader 等价实现

```python
def numpy_dataloader(x, y, batch_size, shuffle=True):
    """
    torch.utils.data.DataLoader 的 NumPy 简化版
    """
    N = len(x)
    indices = np.arange(N)
    if shuffle:
        np.random.shuffle(indices)

    for start in range(0, N, batch_size):
        batch_idx = indices[start:start + batch_size]
        yield x[batch_idx], y[batch_idx]
```
