# 4. 线性代数

NumPy 的 `np.linalg` 模块提供了丰富的线性代数操作，是理解神经网络底层计算（矩阵乘法、特征值、SVD）的基础。

## 4.1 矩阵乘法

```python
import numpy as np

a = np.random.randn(3, 4)
b = np.random.randn(4, 5)

# 三种等价写法
c = np.dot(a, b)            # (3, 4) × (4, 5) → (3, 5)
c = a @ b                   # Python 3.5+ 推荐写法
c = np.matmul(a, b)         # 严格矩阵乘法

# 内积（向量点积）
v1 = np.array([1, 2, 3])
v2 = np.array([4, 5, 6])
dot = np.dot(v1, v2)        # 1*4 + 2*5 + 3*6 = 32
dot = v1 @ v2               # 等价
```

## 4.2 逐元素乘法 vs 矩阵乘法

```python
a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6], [7, 8]])

# 逐元素乘法（Hadamard 积）
c = a * b       # [[5, 12], [21, 32]]

# 矩阵乘法
c = a @ b       # [[19, 22], [43, 50]]
```

**记法**：`*` 是逐元素，`@` / `np.dot` / `np.matmul` 是矩阵乘法。

## 4.3 常用线性代数操作

```python
# 范数
x = np.array([3, 4])
l2 = np.linalg.norm(x)         # L2 范数 = 5
l1 = np.linalg.norm(x, ord=1)  # L1 范数 = 7

m = np.random.randn(3, 4)
l2_rows = np.linalg.norm(m, axis=1)  # 每行的 L2 范数

# 矩阵的逆
a = np.array([[1, 2], [3, 4]])
inv_a = np.linalg.inv(a)
print(a @ inv_a)  # 接近单位矩阵

# 行列式
det = np.linalg.det(a)  # -2.0

# 矩阵的秩
rank = np.linalg.matrix_rank(a)

# 迹（对角线元素和）
trace = np.trace(a)  # 1 + 4 = 5

# 求解线性方程组 Ax = B
A = np.array([[3, 1], [1, 2]])
B = np.array([9, 8])
x = np.linalg.solve(A, B)  # 解 [2, 3]
```

## 4.4 特征值与特征向量

```python
a = np.array([[4, 2], [1, 3]])

eigenvalues, eigenvectors = np.linalg.eig(a)

print("特征值:", eigenvalues)
print("特征向量:")
print(eigenvectors)

# 验证：A v = λ v
v = eigenvectors[:, 0]
lam = eigenvalues[0]
print("A @ v:", a @ v)
print("λ * v:", lam * v)
```

## 4.5 奇异值分解（SVD）

SVD 在降维、推荐系统和 PCA 中有广泛应用。

```python
a = np.random.randn(4, 3)

U, S, Vt = np.linalg.svd(a, full_matrices=False)

print("U:", U.shape)      # (4, 3)
print("S:", S.shape)      # (3,) — 奇异值
print("Vt:", Vt.shape)    # (3, 3)

# 重构：a ≈ U @ diag(S) @ Vt
reconstructed = U @ np.diag(S) @ Vt
print("重构误差:", np.linalg.norm(a - reconstructed))
```

## 4.6 矩阵分解速查

| 分解 | 函数 | 用途 |
|---|---|---|
| SVD | `np.linalg.svd` | PCA、降维、矩阵压缩 |
| QR 分解 | `np.linalg.qr` | 最小二乘、正交化 |
| Cholesky | `np.linalg.cholesky` | 正定矩阵分解 |
| 特征值分解 | `np.linalg.eig` | 主成分分析、谱聚类 |

## 4.7 补充技巧

```python
# 扩充维度方便矩阵乘法
a = np.random.randn(100, 10)           # (100, 10)
b = np.random.randn(100)               # (100,)
c = a.T @ b                            # (10,) — 等价于 (10,100) @ (100,)

# 批量矩阵乘法
a = np.random.randn(5, 3, 4)           # 5 个 3×4 矩阵
b = np.random.randn(5, 4, 2)           # 5 个 4×2 矩阵
c = np.matmul(a, b)                    # (5, 3, 2) — 自动批量
c = a @ b                              # 等价

# Einsum：爱因斯坦求和（万能公式）
a = np.random.randn(3, 4)
b = np.random.randn(4, 5)
c = np.einsum('ik,kj->ij', a, b)      # 矩阵乘法
c = np.einsum('ii->i', a)             # 对角线
c = np.einsum('ij->', a)              # 全部求和
```
