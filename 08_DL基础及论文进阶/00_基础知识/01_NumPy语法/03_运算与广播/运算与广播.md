# 3. 运算与广播

NumPy 的逐元素运算和广播机制是深度学习框架的底层依赖。广播让不同形状的数组能以"合理"的方式对齐运算，无需显式复制数据。

## 3.1 逐元素运算

```python
import numpy as np

a = np.array([[1, 2], [3, 4]])
b = np.array([[5, 6], [7, 8]])

# 四则运算
c = a + b      # 加法
c = a - b      # 减法
c = a * b      # 逐元素乘法（不是矩阵乘法！）
c = a / b      # 除法

# 数学函数（逐元素）
c = np.sqrt(a)       # 平方根
c = np.exp(a)        # e^x
c = np.log(a)        # 自然对数
c = np.abs(a - b)    # 绝对值

# 三角函数
c = np.sin(a)
c = np.cos(a)

# 比较运算
mask = a > 2         # 返回布尔数组
mask = (a > 1) & (b < 8)   # 逐元素逻辑与

# 四舍五入
c = np.round([1.23, 2.67], decimals=1)  # [1.2, 2.7]
c = np.floor([1.7, 2.3])                # [1., 2.]
c = np.ceil([1.3, 2.7])                 # [2., 3.]
```

## 3.2 广播机制

广播是指 NumPy 在算术运算中自动扩展维度不一致的数组，使其 shape 兼容。规则（从后往前比较）：

1. 如果两个维度的长度相等，则兼容。
2. 如果其中一个为 1，则广播（复制）为另一个的长度。
3. 如果不满足 1 或 2，报错。

```python
# 示例一：(3, 4) + (4,) → (3, 4) + (1, 4) → (3, 4)
a = np.ones((3, 4))
b = np.array([1, 2, 3, 4])   # shape (4,)
print(a + b)  # 每行都加 [1,2,3,4]

# 示例二：(3, 4) + (3, 1) → (3, 4)
a = np.ones((3, 4))
b = np.array([[1], [2], [3]])  # shape (3, 1)
print(a + b)  # 每列加对应的值

# 示例三：(3, 4) + (3, 1) + (4,) → (3, 4)
a = np.ones((3, 4))
b = np.array([[1], [2], [3]])   # (3, 1)
c = np.array([10, 20, 30, 40]) # (4,)
print(a + b + c)

# 示例四：不兼容 → 报错
# a = np.ones((3, 4))
# b = np.array([1, 2, 3])       # (3,) — 从后往前比：4 vs 3 不兼容
# a + b  # ValueError
```

**图示理解**：

```
(3, 4)
(   4)  →  广播为 (1, 4) → (3, 4)  ✅

(3, 4)
(3, 1)  →  广播为 (3, 4)          ✅

(3, 4)
(   3)  →  广播为 (1, 3) → (3, 4) ❌  (4 ≠ 3)
```

## 3.3 通用函数（ufunc）

NumPy 的许多函数都是 ufunc（universal function），支持逐元素操作和自动广播。

```python
# ufunc 的 out 参数（原地写入，省内存）
x = np.arange(5)
y = np.empty(5)
np.multiply(x, 2, out=y)  # y = x * 2

# reduce：沿轴归约
x = np.arange(1, 5)
print(np.add.reduce(x))           # 1+2+3+4 = 10
print(np.multiply.reduce(x))      # 1*2*3*4 = 24

# accumulate：沿轴累积
print(np.add.accumulate(x))       # [1, 3, 6, 10]
print(np.multiply.accumulate(x))  # [1, 2, 6, 24]
```

## 3.4 向量化思想

NumPy 的核心优势是向量化——用数组运算替代 Python 循环。

```python
# ❌ 慢：Python 循环
a = np.random.randn(1000000)
b = np.random.randn(1000000)
c = np.empty(1000000)
for i in range(len(a)):
    c[i] = a[i] + b[i]

# ✅ 快：向量化运算
c = a + b   # 底层 C/Fortran 实现，快 50-100 倍
```

**为什么快**：NumPy 的数组运算在底层用 C 实现，且数据在内存中连续排列（contiguous），CPU 缓存友好。
