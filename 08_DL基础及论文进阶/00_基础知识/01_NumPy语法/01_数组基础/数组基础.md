# 1. 数组基础

NumPy 的核心是 `ndarray`（N-dimensional array），一个同构多维数组对象。可以把它理解为 PyTorch `Tensor` 的"前辈"——它们在概念和 API 上高度相似。

## 1.1 创建数组

```python
import numpy as np

# 从列表创建
a = np.array([1, 2, 3])
print(a)  # [1 2 3]

# 指定 dtype
a = np.array([1, 2, 3], dtype=np.float32)

# 全零 / 全一
zeros = np.zeros((3, 4))              # shape (3, 4)
ones = np.ones((2, 3))                # shape (2, 3)

# 单位矩阵
eye = np.eye(4)                       # 4x4

# 指定值填充
full = np.full((2, 3), 7.0)           # 全部填 7

# 等差 / 等分
arange = np.arange(0, 10, 2)          # [0 2 4 6 8]
linspace = np.linspace(0, 1, 5)       # [0.   0.25 0.5  0.75 1.  ]

# 未初始化（内容随机，仅作占位）
empty = np.empty((2, 3))
```

## 1.2 数组属性

```python
x = np.random.randn(2, 3, 4)

print(x.shape)      # (2, 3, 4)
print(x.dtype)      # float64（NumPy 默认双精度）
print(x.ndim)       # 3 — 维度数
print(x.size)       # 24 — 元素总数
print(x.itemsize)   # 8 — 每个元素字节数
print(x.nbytes)     # 192 — 总字节数
```

**常用 dtype**：`np.float32`、`np.float64`（默认）、`np.int32`、`np.int64`、`np.bool_`。深度学习通常用 `np.float32` 以节省内存。

## 1.3 形状变换

```python
x = np.random.randn(4, 4)

# reshape：返回新视图（可能共享内存）
y = x.reshape(16)              # 展平
y = x.reshape(2, 8)            # 改为 (2, 8)
y = x.reshape(-1, 2)           # 自动推断第一维 → (8, 2)

# 展平的三种写法
a = x.reshape(-1)
a = x.ravel()                  # 返回视图（最快）
a = x.flatten()                # 返回拷贝

# 增删维度
y = x[np.newaxis, :, :]       # shape: (1, 4, 4)
y = np.expand_dims(x, axis=0)  # 等价
y = np.expand_dims(x, axis=-1) # shape: (4, 4, 1)
y = np.squeeze(y)              # 去掉所有大小为 1 的维度

# 转置
y = x.T                        # 二维转置
y = np.transpose(x, (1, 0))    # 任意轴交换
y = x.swapaxes(0, 1)           # 交换两个轴
```

## 1.4 拼接与堆叠

```python
a = np.array([[1, 2], [3, 4]])  # (2, 2)
b = np.array([[5, 6]])          # (1, 2)

# 沿第 0 维拼接
c = np.concatenate([a, b], axis=0)  # (3, 2)

# 垂直堆叠（axis=0 的简写）
c = np.vstack([a, b])

# 水平堆叠（axis=1）
d = np.hstack([a, a])               # (2, 4)

# 创建新轴堆叠
e = np.stack([a, a], axis=0)        # (2, 2, 2)

# 切分
x = np.arange(12).reshape(3, 4)
parts = np.split(x, 3, axis=0)      # 切成 3 份 → 每份 (1, 4)
```

## 1.5 类型转换

```python
x = np.array([1.7, 2.3, 3.9], dtype=np.float64)

y = x.astype(np.float32)       # float64 → float32
y = x.astype(np.int32)         # 1.7 → 1（直接截断）
```

## 1.6 副本与视图

这是面试高频考点。NumPy 中有些操作返回视图（与原数组共享内存），有些返回副本。

```python
x = np.array([1, 2, 3, 4])

# 视图（浅拷贝）：修改 y 会影响 x
y = x[1:3]          # 切片 → 视图
y = x.reshape(2, 2) # reshape → 可能为视图
y = x.ravel()       # ravel → 视图（大多数情况）

# 副本（深拷贝）：修改 y 不影响 x
y = x[[1, 2]]       # 花式索引 → 副本
y = x.flatten()     # flatten → 总是副本
y = x.copy()        # 显式拷贝
```

**判断是否共享内存**：`np.shares_memory(x, y)` 返回 `True` 则是视图；`y.base is x` 返回 `True` 也是视图。

## 1.7 排序与搜索

```python
x = np.array([3, 1, 4, 1, 5, 9, 2])

# 排序（返回新数组）
sorted_x = np.sort(x)

# 原地排序
x.sort()

# 返回排序后的下标
idx = np.argsort(x)         # 升序排列的下标

# 最大值 / 最小值下标
max_idx = np.argmax(x)
min_idx = np.argmin(x)

# 沿某个轴
m = np.random.randn(3, 4)
row_max_idx = np.argmax(m, axis=1)  # 每行最大值的列下标
```
