# 5. 随机数与统计

NumPy 的随机模块用于生成训练数据、初始化参数、模拟分布。统计函数用于数据分析、特征工程和模型评估。

## 5.1 随机数生成器

NumPy 1.17+ 推荐使用 `default_rng` 接口（新），但旧 `np.random.*` 接口（旧）在日常代码中仍大量出现。

```python
import numpy as np

# === 推荐写法（新 API） ===
rng = np.random.default_rng(seed=42)

# 均匀分布 [0, 1)
rng.random((3, 4))

# 标准正态分布 N(0, 1)
rng.standard_normal((3, 4))

# 正态分布 N(μ, σ)
rng.normal(loc=0.0, scale=1.0, size=(3, 4))

# 整数
rng.integers(0, 10, size=(5,))

# 随机排列
arr = np.arange(10)
rng.shuffle(arr)                # 原地打乱
permuted = rng.permutation(10)  # 返回新数组

# 按概率抽样
choices = rng.choice([1, 2, 3, 4], size=2, p=[0.1, 0.2, 0.3, 0.4])
```

## 5.2 旧版随机数 API（速查）

```python
# 日常代码中仍常见，了解即可
np.random.seed(42)               # 设置全局种子

np.random.random((3, 4))        # [0, 1) 均匀
np.random.rand(3, 4)            # 同上（简写）
np.random.randn(3, 4)           # 标准正态
np.random.normal(0, 1, (3, 4))  # 正态
np.random.randint(0, 10, (5,))  # 整数
np.random.shuffle(arr)           # 原地打乱
np.random.permutation(10)       # 排列
np.random.choice([1,2,3], size=2, p=[0.1,0.3,0.6])
```

## 5.3 统计函数

```python
x = np.random.randn(1000)

# 描述统计
mean = np.mean(x)            # 均值
std = np.std(x)              # 标准差（默认 ddof=0，总体）
std_sample = np.std(x, ddof=1)  # 样本标准差
var = np.var(x)              # 方差
med = np.median(x)           # 中位数
p25, p75 = np.percentile(x, [25, 75])  # 四分位数

# 沿轴统计（深度学习中最常用）
m = np.random.randn(32, 10)  # 模拟 32 个 batch × 10 个特征

mu = np.mean(m, axis=0)      # 每列均值 → shape (10,)
sigma = np.std(m, axis=0)    # 每列标准差 → shape (10,)
keepdims = np.mean(m, axis=0, keepdims=True)  # shape (1, 10) — 便于广播

# 标准化（Z-score）
normalized = (m - mu) / sigma
```

## 5.4 聚合函数速查

```python
x = np.random.randn(3, 4)

x.sum()            # 总和
x.sum(axis=0)      # 沿第 0 维求和 → shape (4,)
x.min()            # 最小值
x.max()            # 最大值
x.argmin()         # 最小值下标
x.argmax()         # 最大值下标
x.cumsum(axis=1)   # 沿轴累积和
x.prod()           # 所有元素乘积
np.quantile(x, 0.5)  # 中位数（另一种写法）
```

## 5.5 相关性

```python
# 协方差矩阵
cov = np.cov(m, rowvar=False)  # 每列为一个变量

# 相关系数矩阵
corr = np.corrcoef(m, rowvar=False)

# 直方图
counts, bins = np.histogram(x, bins=50)
```

## 5.6 集合运算

```python
a = np.array([1, 2, 3, 4, 5])
b = np.array([3, 4, 5, 6, 7])

np.intersect1d(a, b)   # 交集: [3, 4, 5]
np.union1d(a, b)        # 并集: [1, 2, 3, 4, 5, 6, 7]
np.setdiff1d(a, b)      # 差集 a-b: [1, 2]
np.unique(a)            # 去重
np.in1d(a, b)           # a 中每个元素是否在 b 中: [F,F,T,T,T]
```
