# 模型层实现总览

模型层在 `nanovllm/models/` 和 `nanovllm/layers/` 下，实现了从底层操作（线性变换、注意力、归一化）到高层模型架构（Qwen3）的完整栈。

## 层次结构

```
Qwen3ForCausalLM                        ← 顶层：模型+LM Head
├── Qwen3Model                          ← 中层：Embed + N×DecoderLayer + Norm
│   ├── VocabParallelEmbedding          ← 嵌入（支持 TP）
│   ├── Qwen3DecoderLayer × N           ← Decoder 层
│   │   ├── RMSNorm (input)            ← 输入归一化（含 fused residual add）
│   │   ├── Qwen3Attention             ← 自注意力
│   │   │   ├── QKVParallelLinear      ← QKV 投影（支持 TP）
│   │   │   ├── RMSNorm (q_norm, k_norm)
│   │   │   ├── RotaryEmbedding (RoPE)
│   │   │   ├── Attention              ← FlashAttention + KV-Cache
│   │   │   └── RowParallelLinear      ← 输出投影（支持 TP）
│   │   ├── RMSNorm (post_attn)
│   │   └── Qwen3MLP                   ← FFN
│   │       ├── MergedColumnParallelLinear  ← gate+up 投影（支持 TP）
│   │       ├── SiluAndMul             ← SiLU 门控激活
│   │       └── RowParallelLinear      ← down 投影（支持 TP）
│   └── RMSNorm (final)
└── ParallelLMHead                      ← LM Head（支持 TP）
    └── 继承自 VocabParallelEmbedding
```

## 文件清单

| 文件 | 内容 | 行数 |
|------|------|------|
| `models/qwen3.py` | Qwen3 完整模型实现 | ~220 |
| `layers/linear.py` | 5 种并行线性层 | ~145 |
| `layers/attention.py` | FlashAttention + Triton store kernel | ~75 |
| `layers/embed_head.py` | 并行 Embedding + LM Head | ~60 |
| `layers/layernorm.py` | RMSNorm（+ fused residual） | ~35 |
| `layers/rotary_embedding.py` | RoPE 旋转位置编码 | ~45 |
| `layers/activation.py` | SiLU Gate | ~8 |
| `layers/sampler.py` | Temperature sampling | ~12 |

详细分析见各子页面。
