# 3. 模型定义

PyTorch 中所有模型都继承自 `nn.Module`，需要实现 `__init__` 和 `forward` 两个方法。

## 3.1 nn.Module 基础

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class MyMLP(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim):
        super().__init__()
        # 定义层（带可学习参数的一般放 __init__ 中）
        self.fc1 = nn.Linear(in_dim, hidden_dim)
        self.fc2 = nn.Linear(hidden_dim, out_dim)
        self.dropout = nn.Dropout(0.2)

    def forward(self, x):
        # 定义前向传播过程
        x = F.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.fc2(x)
        return x

model = MyMLP(784, 256, 10)
print(model)
```

**核心原则**：`__init__` 中定义子模块和参数，`forward` 中定义计算流程。PyTorch 会自动追踪 `__init__` 中的 `nn.Module` 子对象和 `nn.Parameter`。

## 3.2 常用层速查

```python
# 全连接
fc = nn.Linear(in_features=128, out_features=64)

# 卷积
conv2d = nn.Conv2d(in_channels=3, out_channels=64,
                   kernel_size=3, stride=1, padding=1)
conv1d = nn.Conv1d(in_channels=16, out_channels=32, kernel_size=3)

# 转置卷积（上采样）
conv_trans = nn.ConvTranspose2d(64, 32, kernel_size=4, stride=2, padding=1)

# 池化
maxpool = nn.MaxPool2d(kernel_size=2, stride=2)
avgpool = nn.AvgPool2d(kernel_size=2)
gap = nn.AdaptiveAvgPool2d((1, 1))  # 全局平均池化 → 输出 (B,C,1,1)

# RNN 系列
rnn = nn.RNN(input_size=10, hidden_size=20, num_layers=2, batch_first=True)
lstm = nn.LSTM(10, 20, 2, batch_first=True, bidirectional=True)
gru = nn.GRU(10, 20, 2, batch_first=True)

# 归一化
bn = nn.BatchNorm2d(64)              # CNN 用
ln = nn.LayerNorm(256)               # Transformer 用
gn = nn.GroupNorm(num_groups=8, num_channels=64)

# Dropout
dropout = nn.Dropout(p=0.5)          # 全连接用

# Embedding
embed = nn.Embedding(num_embeddings=10000, embedding_dim=256)
```

## 3.3 激活函数

`nn` 模块中的激活函数是有状态对象（一般不需要状态）；`F` 中的是纯函数。在模型定义中推荐用 `F.relu()` 等，写在 `forward` 中。

```python
# 常用激活函数（F 中调用）
F.relu(x)
F.leaky_relu(x, negative_slope=0.01)
F.gelu(x)              # Transformer 常用
F.sigmoid(x)
F.tanh(x)
F.softmax(x, dim=-1)

# nn 中调用（作为子模块）
self.relu = nn.ReLU()
self.gelu = nn.GELU()
```

## 3.4 Sequential 容器

```python
# 简单网络用 Sequential 更简洁
cnn = nn.Sequential(
    nn.Conv2d(3, 32, 3, padding=1),
    nn.ReLU(),
    nn.MaxPool2d(2),
    nn.Conv2d(32, 64, 3, padding=1),
    nn.ReLU(),
    nn.MaxPool2d(2),
    nn.Flatten(),
    nn.Linear(64 * 8 * 8, 10)
)
```

## 3.5 自定义层（带可学习参数）

```python
class MyLinear(nn.Module):
    def __init__(self, in_features, out_features):
        super().__init__()
        # 用 nn.Parameter 包装，告诉 autograd 这是可学习参数
        self.weight = nn.Parameter(torch.randn(out_features, in_features))
        self.bias = nn.Parameter(torch.zeros(out_features))

    def forward(self, x):
        return x @ self.weight.t() + self.bias
```

## 3.6 模型参数管理

```python
model = MyMLP(784, 256, 10)

# 查看所有参数
for name, param in model.named_parameters():
    print(f"{name}: {param.shape}")

# 统计参数量
total = sum(p.numel() for p in model.parameters())
trainable = sum(p.numel() for p in model.parameters() if p.requires_grad)
print(f"总参数: {total:,}, 可训练: {trainable:,}")

# 冻结 / 解冻参数
for param in model.fc1.parameters():
    param.requires_grad = False

# 获取特定层的参数
fc1_weights = model.fc1.weight
fc1_bias = model.fc1.bias
```
