# 2. 索引与切片

NumPy 提供三种索引方式：基本切片、花式索引、布尔索引。这也是 PyTorch 张量的索引基础。

## 2.1 基本切片

```python
import numpy as np
x = np.arange(60).reshape(3, 4, 5)  # shape: (3, 4, 5)

# 取第 0 个 batch
print(x[0].shape)          # (4, 5)
print(x[0, :, :].shape)    # (4, 5)

# 取所有 batch 的第 1 行
print(x[:, 1, :].shape)    # (3, 5)

# 取最后一个元素
print(x[..., -1].shape)    # (3, 4)

# 步长切片
print(x[:, ::2, ::2].shape)# (3, 2, 3)

# 负索引
print(x[-1, -2:, -3:].shape)
```

## 2.2 花式索引（整数数组索引）

```python
x = np.arange(10)

# 用整数列表索引
indices = [1, 3, 5, 7]
print(x[indices])  # [1 3 5 7]

# 多维花式索引
m = np.arange(20).reshape(4, 5)
# 取 (0,0), (1,1), (2,2), (3,3) 四个元素
rows = np.array([0, 1, 2, 3])
cols = np.array([0, 1, 2, 3])
print(m[rows, cols])  # [0 6 12 18]

# 行列独立索引（取行 0,2 和列 1,3 的交叉区域）
print(m[[0, 2]][:, [1, 3]])
```

## 2.3 布尔索引

```python
x = np.random.randn(4, 4)

# 筛选所有 > 0 的元素
mask = x > 0
print(x[mask])           # 一维数组，只包含正数

# 按行筛选：取第 0 列 > 0 的所有行
row_mask = x[:, 0] > 0
print(x[row_mask])

# 多条件组合
mask = (x > -1) & (x < 1)  # 注意用 & 不是 and
print(x[mask])

# 原地修改
x[x < 0] = 0               # 把所有负值改为 0
```

## 2.4 `np.where` 的三种用法

```python
# 用法一：只给条件 → 返回满足条件的下标
x = np.array([1, 2, 3, 4, 5])
idx = np.where(x > 3)
print(idx)  # (array([3, 4]),)

# 用法二：条件 + 两个值 → 三目选择
y = np.where(x > 3, 1, 0)  # >3 的位置填 1，否则填 0
print(y)  # [0 0 0 1 1]

# 用法三：多维
m = np.random.randn(3, 4)
rows, cols = np.where(m > 0)
print(list(zip(rows, cols)))  # 所有正数的坐标
```

## 2.5 高级索引技巧

```python
# take：沿某个轴用下标取元素（比花式索引快）
x = np.arange(10)
idx = np.array([2, 5, 7])
print(np.take(x, idx))  # [2 5 7]

# put：原地按索引赋值
np.put(x, idx, -1)       # x[2]=x[5]=x[7]=-1

# diagonal：取对角线
m = np.arange(9).reshape(3, 3)
print(np.diag(m))        # [0 4 8]
print(np.diag(m, k=1))   # [1 5]（上对角线）

# tril / triu：取上下三角矩阵
print(np.tril(m))        # 下三角
print(np.triu(m, k=1))   # 严格上三角
```
