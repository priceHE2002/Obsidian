  ## 第一部分：基本函数

### 1. 幂函数

$$
\int x^k dx = \frac{1}{k+1}x^{k+1} + C \quad (k \neq -1)
$$

- **特殊情况:**
  $$
  \int \frac{1}{x^2} dx = -\frac{1}{x} + C
  $$
  $$
  \int \frac{1}{\sqrt{x}} dx = 2\sqrt{x} + C
  $$

### 2. 倒数函数

$$
\int \frac{1}{x} dx = \ln|x| + C
$$

### 3. 指数函数

$$
\int e^x dx = e^x + C
$$
$$
\int a^x dx = \frac{a^x}{\ln a} + C \quad (a>0 \text{ 且 } a \neq 1)
$$

### 4. 三角函数 (基础)

$$
\int \sin x \,dx = -\cos x + C
$$
$$
\int \cos x \,dx = \sin x + C
$$
$$
\int \tan x \,dx = -\ln|\cos x| + C
$$
$$
\int \cot x \,dx = \ln|\sin x| + C
$$
$$
\int \sec x \,dx = \ln|\sec x + \tan x| + C
$$
$$
\int \csc x \,dx = \ln|\csc x - \cot x| + C
$$

### 5. 三角函数 (进阶)

$$
\int \sec^2 x \,dx = \tan x + C
$$
$$
\int \csc^2 x \,dx = -\cot x + C
$$
$$
\int \sec x \tan x \,dx = \sec x + C
$$
$$
\int \csc x \cot x \,dx = -\csc x + C
$$

## 第二部分：含二次项的积分

### 6. 反正切形式

$$
\int \frac{1}{1+x^2} dx = \arctan x + C
$$
$$
\int \frac{1}{a^2+x^2} dx = \frac{1}{a}\arctan\frac{x}{a} + C \quad (a>0)
$$

### 7. 反正弦形式

$$
\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C
$$
$$
\int \frac{1}{\sqrt{a^2-x^2}} dx = \arcsin\frac{x}{a} + C \quad (a>0)
$$

### 8. 对数形式 (分母为二次)

$$
\int \frac{1}{x^2-a^2} dx = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C
$$
$$
\int \frac{1}{a^2-x^2} dx = \frac{1}{2a}\ln\left|\frac{a+x}{a-x}\right| + C
$$

### 9. 对数形式 (分母含根式)

$$
\int \frac{1}{\sqrt{x^2+a^2}} dx = \ln(x+\sqrt{x^2+a^2}) + C
$$
$$
\int \frac{1}{\sqrt{x^2-a^2}} dx = \ln|x+\sqrt{x^2-a^2}| + C
$$

### 10. 含根式 $\sqrt{a^2-x^2}$ 的积分

$$
\int \sqrt{a^2-x^2} dx = \frac{a^2}{2}\arcsin\frac{x}{a} + \frac{x}{2}\sqrt{a^2-x^2} + C \quad (a>0)
$$

## 第三部分：三角函数平方的积分

*利用三角恒等式进行降幂*

### 11. 正弦/余弦平方

$$
\int \sin^2 x \,dx = \frac{x}{2} - \frac{\sin 2x}{4} + C \quad (\text{使用 } \sin^2 x = \frac{1-\cos 2x}{2})
$$
$$
\int \cos^2 x \,dx = \frac{x}{2} + \frac{\sin 2x}{4} + C \quad (\text{使用 } \cos^2 x = \frac{1+\cos 2x}{2})
$$

### 12. 正切/余切平方

$$
\int \tan^2 x \,dx = \tan x - x + C \quad (\text{使用 } \tan^2 x = \sec^2 x - 1)
$$
$$
\int \cot^2 x \,dx = -\cot x - x + C \quad (\text{使用 } \cot^2 x = \csc^2 x - 1)
$$
